### 选择你的模型 

打开Alter后，只需输入`/`即可显示可用型号并选择您要使用的型号。


#### 选择最喜欢的型号

用鼠标选择模型旁边的星形图标，将该模型标记为您的最爱。
！[收藏模特](/images/models/favorite-model.webp)


使用 API 密钥或本地模型时，它们会显示在顶部的 **自定义** 下。  
所有其他交互均通过 Alter 云进行路由。

### 使用 Alter 云

默认情况下， Alter 使用我们的内部模型路由器 `/best` 设置，根据您的查询选择最佳模型。

！[路由器和型号示例](/images/settings/router.png)


您可以在 **设置 > 默认值 > 模型** 中更改默认模型。

### 使用您的 API 密钥

！[定制本地](/images/models/custom-api-keys.webp)

您可以使用自己的 API 密钥直接连接到提供商的端点：  
1. 转至 **设置 > API 密钥 > 自定义提供商**。  
2. 启用自定义提供程序
3. 在列表中选择您的提供商

> 连接后，**自定义端点** 旁边会出现一个勾号。  

您的提供商的型号将列在 **自定义** 部分下。

> 我们仅支持 OpenAI 兼容端点，这意味着您无法使用 Anthropic API 密钥。

以下是基于提供的图像的自定义密钥支持的平台列表：

1.蔚蓝
2.Gemini
3.LM Studio
4. 米斯特拉尔
5.Ollama
6.OpenAI
7. 打开路由器
8. 自定义（任何 OpenAI 兼容端点）

#### 在 Alter 中设置 Azure 端点

1. **启用自定义提供程序：** 在Alter的设置中，启用“自定义提供程序”选项。
2. **选择 Azure：** 选择 Azure 作为您的提供商。
3. **输入完整聊天完成端点 URL：** 提供完整聊天完成端点 URL，包括 URL 中的 `api-version` 参数和型号名称。  例如：`https://yourinstance.openai.azure.com/openai/deployments/your-model-name/chat/completions?api-version=2024-02-15-preview`
4. **输入 API 密钥：** 输入您的 Azure OpenAI API 密钥。

重要考虑因素

* **需要完整端点：** 确保使用 *完整* 聊天完成端点，因为 Alter 从 URL 中提取模型名称。
* **URL 中的模型名称：** 模型部署名称*必须*包含在 URL 路径中。
* **API 版本：** URL *必须*包含 `api-version` 参数。
