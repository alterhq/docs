### 使用您的 API 密钥

！[定制本地](/images/models/custom-api-keys.webp)

您可以使用自己的API直接连接到顶部的端点：  
1. 转至**设置 > API核心 > 自定义工厂**。  
2. 实现自定义提供程序
3. 在列表中选择您的事业

> 连接后，**自定义端点** 旁边会出现一个勾号。  

您的提供商的型号将列在 **自定义** 部分下。

> 我们仅支持 OpenAI 兼容端点，这意味着您无法使用 Anthropic API 密钥。

以下是基于提供的图像的自定义密钥支持的平台列表：

2.双子座

5.奥拉马
6.开放人工智能
7.打开路由器
8. 自定义（任何 OpenAI 兼容端点）

#### 在 Alter 中设置 Azure 端点

1. **启用自定义提供程序：** 在Alter的设置中，启用“自定义提供程序”选项。
2. **选择Azure：**选择Azure作为您的事业。
3. **输入完整的聊天完成端点 URL：** 提供完整的聊天完成端点 URL，包括 URL 中的 `api-version` 参数和型号名称。 例如：`https://yourinstance.openai.azure.com/openai/deployments/your-model-name/chat/completions?api-version=2024-02-15-preview`
4. **输入API密钥：**输入您的Azure OpenAI API密钥。

重要考虑因素

* **需要完整端点：**确保使用 *完整* 聊天完成端点，因为改变从 URL 中提取模型名称。
* **URL 中的模型名称：** 模型部署名称*必须*包含在 URL 路径中。
* **API 版本：** URL *必须*包含 `api-version` 参数。
