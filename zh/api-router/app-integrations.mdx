## 集成检查清单

1.将基本URL设为`https://alterhq.com/api`（若需要使用`/v1`）
2. 将认证标头设为`Authorization: Bearer YOUR_API_KEY`
3.使用`Provider#Model-name`格式的模型ID
4. 首次请求前先调用一次模型列表
