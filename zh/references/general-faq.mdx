## Alter 与其他Tools有何不同？

Alter 是一款原生 macOS AI 助手，它了解您当前的工作环境并直接与您的应用程序集成。

## 系统要求是什么？

Alter 支持 macOS Ventura 13 或更高版本，并在 Apple Silicon 和 Intel Mac 上运行。

## 哪些 macOS 应用程序与 Appsense 兼容？

所有 macOS 应用程序均与 Alter Appsense 兼容，开箱即用。

## Alter 如何节省我的时间？

Alter 在您的工作流程中提供上下文帮助：编写支持、会议记录和摘要以及拖放文件上下文。

## Alter 可以使用哪些 AI 模型？

Alter 支持领先的提供商并快速添加新发布的模型。

- 支持视觉的型号（例如 Gemini、Claude、GPT-5、Pixtral）
- 以文本为中心的模型（例如 Llama、Qwen、DeepSeek）

## 我可以使用自己的 API 密钥吗？

是的。参见[如何使用您自己的 API 密钥](/how-to/use-byok)。

## 我可以离线使用 Alter 或与本地模型一起使用吗？

是的。 Alter 支持 Ollama 和 LM Studio 进行本地处理。参见[注重隐私的本地模型工作流程](/use-cases/local-model-privacy)。

注意：LM Studio 支持可能因设备类别而异； Ollama 在 Mac 上得到广泛支持。

## 我可以在路由器上使用 Alter 吗？

是的。 Alter 通过在设置中输入端点 + API 密钥来与 OpenAI 兼容的路由器配合使用。
Alter 通过其 API 网关本身也是一个路由器。

## Alter 可以看到我的数据吗？

发送到模型的提示和数据在传输过程中进行加密。
Alter 收集匿名使用情况分析，可以在设置中禁用。
