### 全球导航

跨窗口可用（除非目标窗口已处于活动状态）。

- **Shift + Cmd + H**：打开 Alter 集线器
- **Shift + Cmd + E**：打开动作编辑器
- **Shift + Cmd + T**：打开 Tools Manager
- **Alt + 空格**（默认）：开放式凹口 — 在 **设置 > 默认值 > 开放式凹口键绑定**中进行配置
- **Alter 快速菜单快捷方式**：从任何地方打开 Alter — 可在 **设置 > 常规 > 快速访问** 中进行配置
- **语音转文本热键**：开始语音输入 — 在 **设置 > 听写 > 快捷方式** 中进行配置（默认：Fn 键）

### 快速面板

当快速面板打开时（从凹口处）：

- **←**、**↑**、**→**、**↓**：在对象之间导航
- **输入**：选择一个对象

### 上下文部分

- **←**、**↑**、**→**、**↓**：在上下文之间导航
- **空格键**：添加/删除选定的上下文
- **Cmd + K**：从活动对话中删除所有上下文
- **Cmd + V**：粘贴剪贴板内容以添加到上下文

### 提示框

- **输入**：提示您的活动模型
- **Cmd + Enter**：提示使用网络搜索（通过 Gemini web）
- **Shift + Enter**：换行
- **@**：将元素添加到上下文中
- **##**：显示 Alter 对话的历史记录
- **>**：显示会议转录的历史记录
- **/**：列出并选择您的型号
- **?**：显示帮助菜单

### Alter 中心窗口

- **Cmd + F**：焦点搜索（箭头键导航结果）
- **Cmd + B**：切换侧边栏可见性
- **Cmd + T**：新选项卡（开始新聊天）
- **Cmd + J**：创建Workspaces（需要活动上下文）
- **Cmd + Opt + O**：切换到Workspaces...
- **Cmd + K**：清除所有上下文（需要活动上下文）
- **Cmd + Delete**：删除对话（需要活动对话）
- **Cmd + W**：关闭选项卡（如果是最后一个选项卡，则关闭窗口）
- **Cmd + Opt + W**：关闭所有选项卡（需要 >1 个选项卡）
- **Cmd + Opt + ←**：上一个选项卡（需要 >1 个选项卡）
- **Cmd + Opt + →**：下一个选项卡（需要 >1 个选项卡）
- **Cmd + 1...9**：切换到特定选项卡索引

#### Workspace（需要选定的Workspaces）

没有默认的快捷键；这些可以通过 macOS 系统设置进行设置：

- 打开Workspaces设置
- 重命名Workspaces
- 设置权限
- 切换沙箱
- 启动/停止沙箱容器
- 在沙箱中打开终端

### 动作编辑器窗口

- **Cmd + F**：焦点搜索
- **Cmd + N**：新操作
- **Cmd + D**：重复操作（需要选定的操作）
- **Cmd + E**：导出操作（需要选择操作）
- **Cmd + O**：导入操作
- **Cmd + Delete**：删除操作（需要选择操作）
- **Cmd + W**：关闭窗口
- **Cmd + [**：上一节
- **Cmd + ]**：下一节

### Tools Manager 窗口

- **Cmd + W**：关闭窗口
- **Cmd + F**：焦点搜索
- **选择“技能”时**：Cmd + O（导入技能）、Cmd + R（刷新列表）
- **选择“远程 MCP 服务器”时**：Cmd + N（添加服务器）、Cmd + R（刷新列表）
- **选择“快速操作”时**：Cmd + N（新建快速操作）、Delete（删除快速操作，需要选择）

### 实时会议小组

仅在会议期间可用：

- **Cmd + Shift + ↓**：打开/关闭实时字幕
- **Shift + Cmd + →**：切换实时记事本

### Alter 设置

- **Cmd + ,**：打开设置（当 Alter 窗口可见时）
