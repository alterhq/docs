## 如何打开Alter？

1. 将鼠标悬停在凹口上。
2. 按`Alt+Space`。
3. 按住 `Fn` 并说话。

！[打开Alter](/images/getting-started/open-notch.gif)

有关更多详细信息，请参阅 [入门](/getting-started/getting-started-path)。

## 如何添加自己的 API 密钥？

1. 打开设置 (`Cmd+,`)。
2. 转到 API 密钥。
3. 选择提供商并输入密钥。

！[自定义 API 密钥](/images/models/custom-api-keys.webp)

详细说明请参见[操作方法：使用您自己的 API 密钥](/how-to/use-byok)。

## 如何向提示添加上下文？

- 将文件拖放到凹口上。
- 右键单击​​文件并选择添加到 Alter。
- 使用 `@` 从提示中添加上下文。

！[拖放](/images/context/drag-and-drop.gif)

请参阅 [Alter 窗口](/zh/getting-started/alter-window#the-context-section) 了解更多详细信息。

## 如何快速更改设置？

- 打开 Alter 并按 `Cmd+,`。
- 或打开三点菜单 -> 设置。

！[设置菜单](/images/settings/settings-menu.png)

有关更多详细信息，请参阅 [Alter 设置](/how-to/settings-guide)。

## 如何配置听写？

设置 -> 听写，然后选择处理器、语言和替换。

！[听写](/images/settings/dictation.webp)

请参阅 [听写指南](/workflows/dictation) 了解完整的设置说明。
