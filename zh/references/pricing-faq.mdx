## 计划一览

- **免费（0 美元）**：所有核心 Alter 功能都带有您自己的 API 密钥或本地模型。无需信用卡。
- **Local+（附加）**：高级本地语音模型（Whisper Pro 和 Parakeet V3），最多 3 个设备。
- **专业版（订阅）**：管理对 50 多个云模型、路由器模型（最佳/快速/轻型）、API 网关、远程Tools、优先支持的访问。
- **终身（一次性）**：与订阅计划相同的访问级别，无需定期计费。

## 试用和免费使用

### Alter 真的免费吗？

是的。使用本地模型或您自己的 API 密钥时，Alter 可以免费使用核心功能。

### 7 天试用后会发生什么？

试用后，该应用程序在 BYOK/本地使用情况下仍可正常运行。付费计划消除了升级提示并提供托管模型访问。

## 无限制且合理使用

###“无限”是什么意思？

付费计划包括 Alter 应用程序中的无限请求，但须遵守合理使用政策。

### 合理使用如何运作？

- 使用每月预算系统，没有严格的消息上限。
- 如果超出预算，请求可能会转至具有应用内指示的低成本型号。
- API 网关有每日限制（每天 200 个请求），然后限制为每 10 分钟 1 个请求。
- 政策禁止大量使用自主编码代理流量。

## 路由器

### Alter 光是什么？

通过选择精简/闪存型号来关注速度/成本的路由器。

### Alter 最好是什么？

平衡路由器，选择强大的型号（例如 Sonnet 系列）以提高质量和速度。

### 什么是 Alter 快？

速度优先的路由器，优先考虑非常高吞吐量的模型。

## 设备和许可

### 我可以在多个设备上使用 Alter 吗？

是的，所有计划都有。单个用户一次的活跃使用仅限于一台设备。

### 计划的设备限制

- **本地+**：最多安装 3 个设备
- **专业版/终身**：无限安装
- **所有计划**：一次一台活动设备；没有许可证共享
