在通过“获取活动应用程序上下文”识别目标后，使用此工具执行具体的 UI 交互。

## 支持的操作

- `click`：单击目标元素。
- `fill`：在文本字段或文本区域中输入文本。
- `shortcut`：发送键盘快捷键（例如`cmd+c`、`ctrl+shift+t`）。

## 输入

- `elementID`（`click` 和`fill` 必需）：来自“获取活动应用程序上下文”输出的元素 ID。
- `text`（`fill` 必需）：要插入到目标输入中的值。
- `shortcut`（`shortcut` 必需）：快捷方式字符串。
- `windowID`（可选）：定位特定窗口 (`pid/title`) 而不是活动窗口。

## 注释

- 元素 ID 会随着 UI 更改和会话计时而过期。如果元素不再有效，则重新读取上下文。
- 对于后台窗口定位，请在上下文读取和执行步骤中使用相同的`windowID`。
