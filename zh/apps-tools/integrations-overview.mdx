## 本地工具

Alter 提供**17+ 强大的本地工具**，可直接与您的 Mac 应用程序集成。与其他 AI 助手不同，Alter 实际上通过自然语言控制您的 Mac。

### 如何使用本地工具

<Steps>
  <Step title="打开 Tools 管理器">
    单击 Alter → **工具管理器** 中的三个点
  </Step>
  <Step title="启用Tools">
    在**本地工具**部分中，搜索并启用工具（日历、地图等）
  </Step>
  <Step title="使用自然语言">
    要求 Alter 执行操作：
    - “给约翰发消息说我要迟到了”
    - “在我的日历中添加明天下午 2 点的会议”
    - “显示我的未读电子邮件”
  </Step>
</Steps>

<Note>
  使用工具管理器中的全局开关（设置 > 工具）打开/关闭所有工具
</Note>

## x-callback-url 集成

Alter 支持 [x-callback-url](https://x-callback-url.com/) 协议进行应用间通信。

### x-callback-url 是什么？

允许应用程序使用特殊 URL 相互触发操作的标准。无缝传递数据、执行操作并返回。

### 从其他应用程序触发更改

<Steps>
  <Step title="获取网址">
    在操作编辑器中，选择一个操作 →“URL 回调”部分
  </Step>
  <Step title="复制方案">
    复制该操作的 `alter://` URL
  </Step>
  <Step title="随处使用">
    粘贴到快捷方式、注释、浏览器等中。
  </Step>
</Steps>

**示例：**
```
alter://action/ask-web?input=What+is+Alter+MacOS
```

### 从 Alter 触发其他应用程序

使用目标应用程序 URL 方案创建可点击的链接：

|应用程序 |示例网址 |行动|
|-----|-------------|--------|
|熊| `bear://x-callback-url/create?title=Demo` |创建笔记 |
|事情| `things://add?title=Task` |添加任务 |
|草稿 | `drafts://x-callback-url/create?text=Hello` |创建草稿 |

<Card title="支持的应用程序" icon="check-circle">
  Bear、Things、Drafts、OmniFocus、1Writer、Agenda 以及数百个其他工具
</Card>

<Info>
  请参阅 [URL 回调指南](/workflows/url-callbacks) 以获取完整文档
</Info>
