---
title: "键盘快捷键不起作用"
description: "修复 Alt+Space 或自定义快捷键无法打开 Alter 时的热键问题"
---
<Note>
  **首先，检查基础知识：** Alter 是否实际运行？在屏幕顶部的菜单栏中查找 Alter 图标。
</Note>

##发生了什么事？

您按下键盘快捷键（默认为 `Alt + Space`），但是：
- 什么也没发生
- 快捷菜单不出现
- 它适用于某些应用程序，但不适用于其他应用程序

<AccordionGroup>
  <Accordion icon="gear" title="问题：快捷方式已更改">
    **解决方案：** 检查您的快捷方式实际设置的内容
    
    1.打开Alter（如果需要，点击菜单栏图标）
    2. 按`Cmd + ,` 打开设置
    3. 转到**默认**选项卡
    4. 查看 **打开凹口键绑定** – 这显示了您当前的快捷方式
    5. 按`Alt + Space`（或您喜欢的组合）进行设置
  </Accordion>
  
  <Accordion icon="xmark" title="问题：另一个应用程序正在使用相同的快捷方式">
    **解决方案：** 找到冲突并更改
    
    使用 `Alt + Space` 的常见应用程序：
    ——阿尔弗雷德
    - 光线投射
    - 聚光灯（有时）
    - 语言切换器
    
    **尝试不同的快捷方式：**
    - `Cmd + Shift + A`
    - `Ctrl + Option + Space`
    - `Fn + F12`
    
    选择您可以轻松记住且其他应用程序没有使用的内容。
  </Accordion>
  
  <Accordion icon="lock" title="问题：权限问题">
    **解决方案：** 检查辅助功能权限
    
    Alter 需要辅助权限来检测全局热键：
    
    1. 打开 **系统设置** → **隐私和安全** → **辅助功能**
    2. 确保 **Alter** 在列表中并已启用
    3. 如果呈灰色，请单击锁定图标并输入您的密码
    4. 关闭 Alter，然后再打开
  </Accordion>
</AccordionGroup>

## 快速测试

进行更改后：
1. 单击进入任何应用程序（例如 Safari 或 Notes）
2. 按您的快捷键（`Alt + Space` 或您设置的任何内容）
3. **成功：** Alter 缺口/快速面板应立即出现

<Tip>
  **专业提示：** 您还可以将鼠标悬停在屏幕顶部的凹口上来打开 Alter。这完全绕过了键盘快捷键！
</Tip>
