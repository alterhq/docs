使用本指南选择最适合您的更改网页搜索配置。

<Info>
  先从Alter路由器开始。`Fast`、`Best`、`Fair`、`Light`内置网页搜索，需要额外已配置。
</Info>

## 可用选项

- **本地工具：** `Web Search` 和 `Web Search Gemini`
- **远程MCP服务器：** Tavily、LinkUp、Exa
- **应用程序库：** Firecrawl、LinkUp、Exa（如果您的工作空间可用）
- **更改路由器:** `Fast`、`Best`、`Fair`、`Light`，内置网页搜索

## 服务定位速览

- **Tavily：**面向AI代理的网页访问层，提供搜索、提取、研究、爬行。
- **Firecrawl：**将网站转换为LLM可靠数据，强调抓取和爬行。
- **链接：**面向AI应用的搜索，提供标准与深度模式，并引入来源。
- **Exa:** 强调人工智能搜索 API 的质量，覆盖搜索到研究的多个端点。

## 混合策略

1.先用路由器进行初步搜索，最容易上手且默认效果好。
2.对没有初步搜索能力的模型（包括本地模型），使用内置工具扩展网页搜索，无需在Alter之外注册账号。
3.想在任何模型上获得更强的效果和更高的控制力时，加入MCP吧。
4.多页面提取提取时，再需要加入scraper。

<Warning>
  免费限制和损耗会变化。请在使用前核对最新。
</Warning>

<Tip>
  不要同时启用多个网页搜索工具，这会干扰模型的工具。
</Tip>
