Exa 是面向 AI 的网页搜索 provider。

## Exa 的定位

- "The best search API for AI"
- 强调质量与 benchmark
- 覆盖 search、contents、answer、crawl、research 等 endpoints

## 连接步骤

1. 打开 **Tools Manager**。
2. 在 **Remote MCP Servers** 或 **Apps Gallery** 中添加 Exa。
3. 粘贴 API key。
4. 启用连接器。

## 免费计划

- 常见描述约为每月 **1,000 requests**。

<Warning>
  配置时请核对最新价格与限制。
</Warning>
