Tavily 适合网页研究、信息提取与 crawling。

## Tavily 的定位

- 面向 AI agents 的网页访问层
- 一套 API 覆盖 search、extraction、research、crawling
- 强调基于最新网页内容的 grounding

## 连接步骤

1. 打开 **Tools Manager**。
2. 进入 **Remote MCP Servers**。
3. 添加 Tavily。
4. 粘贴 API key。
5. 启用连接器。
