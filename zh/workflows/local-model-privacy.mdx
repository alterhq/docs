您正在与：
- 客户机密信息
- 医疗记录
- 专有业务数据
- 个人财务文件
- 安全研究

您担心将这些数据发送到 OpenAI 或 Google 等云人工智能服务，即使他们声称这些数据是私有的。

## 本地模型如何工作

您无需将数据发送到互联网，而是将 AI 模型下载到 Mac 上（就像下载大型应用程序一样）并在本地运行。您的数据永远不会离开您的计算机。

## 设置：两个简单的选项

### 选项 1：Ollama（推荐初学者）

Ollama 使运行本地模型就像安装应用程序一样简单。

<Steps>
  <Step title="Install Ollama">
    1. 从[ollama.com](https://ollama.com)下载
    2. 像安装任何 Mac 应用程序一样安装它
    3.它在您的菜单栏中运行
  </Step>
  
<Step title="Download a model">
    打开终端并运行：
    ```bash
    ollama pull llama3.1
    ```
    
这将下载 Meta 的 Llama 3.1 模型（~4.7GB）。其他不错的选择：
    - `ollama pull mistral`（更快，更小）
    - `ollama pull codellama`（非常适合代码）
    - `ollama pull mixtral`（能力更强，更大）
  </Step>
  
<Step title="Connect to Alter">
    1. 打开更改 → 设置 (`Cmd + ,`)
    2. 转到 **API 密钥** 选项卡
    3. 在 **自定义提供商** 下，选择 **Ollama**
    4. 确保 Ollama 正在运行（检查菜单栏）
    5. 打开**启用自定义提供程序**
  </Step>
  
<Step title="Start using it">
    1. Press `/` in Alter's prompt box
    2. 查找 **自定义** 部分
    3. 选择您的本地型号（例如“llama3.1”）
    4. 提出任何问题 – 它完全在您的 Mac 上运行！
  </Step>
</Steps>

### 选项 2：LM Studio（更多控制）

LM Studio 为您提供图形界面来管理模型。

<Steps>
  <Step title="Install LM Studio">
    从 [lmstudio.ai](https://lmstudio.ai) 下载并安装。
  </Step>
  
<Step title="Download a model">
    1.打开LM Studio
    2.浏览型号目录
    3. 下载适合您的需求和 Mac 规格的一款
    4.启动本地服务器（大“启动服务器”按钮）
  </Step>
  
<Step title="Connect to Alter">
    1. 在更改设置 → **API 密钥**
    2. 选择 **LM Studio** 作为提供商
    3. LM Studio默认运行在`localhost:1234`上
    4. 将 **启用自定义提供程序** 打开
  </Step>
</Steps>

## 需要了解的权衡

<AccordionGroup>
  <Accordion icon="gauge" title="Speed vs Privacy">
    **云模型：** 更快、更强大
    **本地模型：** 较慢，但完全私有
    
配备 16GB RAM 的 MacBook Pro 可以流畅运行小型机型。对于较大的型号，您需要 32GB 以上的 RAM。
  </Accordion>
  
<Accordion icon="brain" title="Capability differences">
    本地模型每个月都在变得更好，但云模型（GPT-4，Claude）仍然更能够进行复杂的推理。
    
**最佳方法：** 对敏感数据使用本地模型，对不太敏感的复杂任务使用云模型。
  </Accordion>
  
<Accordion icon="hard-drive" title="Storage requirements">
    型号范围从 4GB 到 70GB+。确保您有足够的磁盘空间。
    
**良好的入门型号：**
    - Llama 3.1 8B (~4.7GB) – 速度快，质量不错
    - Mistral 7B (~4.1GB) – 良好的平衡性
    - Mixtral 8x7B (~26GB) – 更高质量，需要更多 RAM
  </Accordion>
</AccordionGroup>

## 私有工作流程的最佳实践

<Tip>
  **对敏感任务使用特定操作：** 创建配置为始终使用本地模型的更改操作。这样您就不会意外地使用云模型来进行敏感工作。
</Tip>

<Tip>
  **设置隐私工作区：** 仅为敏感文档创建一个工作区。这使它们保持井井有条，并提醒您使用本地模型。
</Tip>

<Tip>
  **验证其是否正常工作：** 设置后，断开 WiFi 并尝试使用 Alter。如果它仍然适用于您的本地模型，那么您就知道它是真正的本地模型。
</Tip>
