LinkUp 连接简单，适用于 standard search 和 deep research。

## LinkUp 的定位

- "World's best search for AI apps"
- 强调带来源的事实型 grounding
- 提供 Standard 和 Deep 两种模式

## 连接步骤

1. 打开 **Tools Manager**。
2. 通过 **Apps Gallery** 或 **Remote MCP Servers** 连接 LinkUp。
3. 粘贴 API key。
4. 启用连接器。

## 免费计划

- 常见描述为 **1,000 standard requests** + **100 deep research requests**。
