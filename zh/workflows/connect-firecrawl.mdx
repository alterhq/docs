Firecrawl 更适合规模化结构化提取。

## Firecrawl 的定位

- "Turn websites into LLM-ready data"
- 面向 scraping、crawling 与 LLM 数据准备
- 强调动态页面和生产可用性

## 连接步骤

1. 打开 **Tools Manager**。
2. 进入 **Apps Gallery**。
3. 连接 **Firecrawl**。
4. 填写 API key。

## 免费计划

- Firecrawl 提供免费方案。

<Warning>
  请核对当前配额、速率限制和数据保留策略。
</Warning>
