### 一般概念

Alter提供了两个与AI交互的主要接口：

<AccordionGroup>
  <Accordion icon="layout-dashboard" title="The Hub">
    主窗口界面带有选项卡、侧边栏和全面的工作区管理。最适合：
    - 扩展对话
    - 工作区管理
    - 具有多个上下文的多任务处理
    - 审查会议记录
  </Accordion>
  
<Accordion icon="message-circle" title="Quick Panel">
    可通过缺口或热键访问轻质浮动面板。非常适合：
    - 快速查询
    - 语音命令
    - 快速添加上下文
    - 快速互动
  </Accordion>
</AccordionGroup>

## 集线器接口

<Frame>
  该中心分为三个主要部分：
  
1. **侧边栏** - 对话、工作区、记录等的导航
  2. **主要内容区域** - 显示活动内容的选项卡式界面
  3. **输入区域** - 提示框和上下文管理
</Frame>

### 打开中心

<Steps>
  <Step title="Keyboard shortcut">
    按`Shift + Cmd + H`
  </Step>
  <Step title="Menu bar">
    从更改菜单栏图标中选择“打开集线器”
  </Step>
  <Step title="From a conversation">
    从浮动面板打开对话链接
  </Step>
</Steps>

### 键盘快捷键

|快捷方式 |行动|
|----------|--------|
| `Cmd + B` |切换侧边栏可见性 |
| `Cmd + W` |关闭活动选项卡 |
| `Cmd + Option + →` |下一个选项卡 |
| `Cmd + Option + ←` |上一个选项卡 |
| `Cmd + K` |清除所有上下文 |
| `Cmd + Delete` |删除活动对话 |

### 管理上下文

<Steps>
  <Step title="Add context">
    拖放文件，或在提示框中使用`@`
  </Step>
  <Step title="Navigate">
    使用箭头键在上下文之间移动
  </Step>
  <Step title="Clear">
    按`Cmd + K`删除所有上下文
  </Step>
</Steps>

### 检查面板

## 快速面板界面

快速面板（可从凹口访问）分为三个部分：

<CardGroup cols={3}>
  <Card title="1. Context" icon="layers">
    您选择的文件、应用程序和参考
  </Card>
  <Card title="2. Prompt" icon="edit">
    您在哪里输入或说出您的请求
  </Card>
  <Card title="3. Additional Selection" icon="plus">
    快速访问模型和操作
  </Card>
</CardGroup>

＃＃＃ 导航

使用快速面板中的键盘快捷键：
- `←`、`↑`、`→`、`↓` - 在对象之间导航
- `Enter` - 选择一个对象
- `@` - 将元素添加到上下文
- `#` - 显示对话历史记录
- `>` - 显示会议历史记录
- `/` - 列出并选择型号
- `?` - 显示帮助菜单

<Note>
  有关更多键盘快捷键，请参阅下面的 **提示框** 部分。
</Note>

### 键盘导航

|关键|行动|
|-----|--------|
| `←` `↑` `→` `↓` |导航上下文 |
| `Space` |添加/删除选定的 |
| `Cmd + K` |删除所有上下文 |

直接粘贴内容：`Cmd + V`

您的剪贴板内容会自动添加到更改的上下文中。

### 连续语音提示

无需触摸键盘即可进行长时间的语音对话：

<Steps>
  <Step title="Hold voice hotkey">
    按住`Fn`（或您配置的键）
  </Step>
  <Step title="Speak naturally">
    大声说出您的提示
  </Step>
  <Step title="Release">
    放开发送至对话
  </Step>
  <Step title="Keep going">
    每个语音提示都会添加到同一个线程
  </Step>
</Steps>

<Tip>
  **免提工作流程：** 非常适合头脑风暴、听写或双手忙碌时。
</Tip>

## 对话历史记录

### 管理聊天记录

通过在提示框中输入`#`或使用Hub的侧边栏来访问历史记录。

<Frame>
  ![带有标签的历史记录](/images/other/history-tags.webp)
</Frame>

### 用标签组织

添加 `#personal` 等标签对聊天进行分类：

```
# #personal #food
```

搜索匹配多个标签的对话。

### 重命名对话

<Steps>
  <Step title="Right-click">
    右键单击列表中的任意对话
  </Step>
  <Step title="Select Rename">
    从菜单中选择`Rename`
  </Step>
  <Step title="Type new name">
    给它一个描述性的名称
  </Step>
</Steps>

### 导出对话

导出为 Markdown 提供文档或共享：

<Steps>
  <Step title="Find conversation">
    在历史记录中找到它
  </Step>
  <Step title="Right-click">
    打开上下文菜单
  </Step>
  <Step title="Select Export">
    选择导出选项
  </Step>
  <Step title="Save">
    选择位置并另存为 .md 文件
  </Step>
</Steps>

<Card title="Export Uses" icon="download">
  - 文档
  - 与团队分享
  - 存档重要讨论
  - 在其他工具中使用
</Card>
