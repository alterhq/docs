使用此页面可以快速为 Alter 提供对您的任务重要的文件和文件夹。

## 从 Finder 或应用程序添加上下文

您可以通过将文件或文件夹拖动到凹口或中心来添加内容。您还可以使用**右键单击 > 添加到更改**。

## 支持的内容

<CardGroup cols={2}>
  <Card title="Documents" icon="file-text">
    `.xlsx`、`.docx`、`.pdf`、`.txt`、`.md`、`.org`、`.rtf`
  </Card>
  <Card title="Images" icon="image">
    `.jpg`、`.png`、`.heic`、`.gif`、`.tiff`、`.svg`
  </Card>
  <Card title="Web content" icon="globe">
    `.html`、URL和拖动的文本
  </Card>
  <Card title="Email" icon="mail">
    `.eml`和Apple Mail消息
  </Card>
  <Card title="Audio" icon="music">
    `.mp3`、`.wav`、`.m4a`、`.aac`、`.flac`
  </Card>
  <Card title="Code" icon="code">
    源文件、配置文件和脚本
  </Card>
</CardGroup>

<Note>
  Alter 支持目录删除、图像和 PDF 的 OCR 以及自动文本编码检测。
</Note>

## 音频和视频转录

添加音频或视频时，Alter会自动开始录制。

<Steps>
  <Step title="Drop one or more files">
    将媒体文件拖至凹口或集线器。
  </Step>
  <Step title="Wait for processing">
    转录自动开始。
  </Step>
  <Step title="Ask targeted questions">
    尝试：“总结关键决策”或“按所有者列出行动项目”。
  </Step>
</Steps>

<Warning>
  对于图像分析，请使用具有视觉功能的模型。
</Warning>
