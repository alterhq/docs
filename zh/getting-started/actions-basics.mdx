当您重复相同的任务模式并想要一致的输出质量时，请使用操作。

## 一个动作会给你带来什么

<CardGroup cols={3}>
  <Card title="Reusable prompts" icon="repeat">
    运行相同的工作流程，无需每次都重写指令。
  </Card>
  <Card title="Model choice" icon="brain">
    为每项任务分配最佳模型。
  </Card>
  <Card title="Tool control" icon="sliders-horizontal">
    仅启用更安全、更快运行所需的工具。
  </Card>
</CardGroup>

## 创建你的第一个动作

<Steps>
  <Step title="Open Action Editor">
    打开更改窗口菜单并选择 **动作编辑器**。
  </Step>
  <Step title="Create or duplicate">
    创建新操作，或在自定义之前复制内置操作。
  </Step>
  <Step title="Set clear instructions">
    定义一项具体结果，例如每周摘要、发行说明或收件箱分类。
  </Step>
  <Step title="Save and run">
    Test with one real input and refine wording if needed.
  </Step>
</Steps>

<Note>
  内置操作是只读的。复制它们以自定义行为。
</Note>

## 选择每个操作的模型和工具

在操作中使用**高级**设置来分配模型和工具访问权限。

- 为敏感任务选择本地模型。
- 为自动化繁重的任务选择更快的模型。
- 从最少的工具开始，然后仅添加您的工作流程所需的工具。
