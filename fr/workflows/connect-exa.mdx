Exa est un provider specialise dans la recherche web, utilisable avec Alter via des integrations de type MCP.

## Comment Exa se positionne

- "The best search API for AI"
- Un positionnement qualite d'abord, avec des benchmarks mis en avant
- Une API unique qui couvre search, contents, answer, crawl et workflows de recherche approfondie
- Un accent sur la latence faible et les controles enterprise

## Ou connecter Exa

- **Remote MCP Servers**
- **Apps Gallery** (si disponible dans votre workspace)

## Connexion Exa

1. Ouvrez **Tools Manager**.
2. Ajoutez Exa dans **Remote MCP Servers** ou connectez-le depuis **Apps Gallery**.
3. Collez votre cle API.
4. Activez le connecteur.

## Plan gratuit

- Exa est souvent decrit avec un plan gratuit autour de **1 000 requetes par mois**.

<Warning>
  Verifiez le pricing et les limites Exa pendant la configuration. Les plans gratuits peuvent changer.
</Warning>

## Exemples de prompts

- "Recherche des articles techniques recents sur cette API et extrais les notes de migration."
