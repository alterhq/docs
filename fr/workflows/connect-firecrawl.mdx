Firecrawl est adapte quand vous avez besoin d'extraction structuree sur de nombreuses pages.

## Comment Firecrawl se positionne

- "Turn websites into LLM-ready data"
- Une API web data orientee developpeurs avec scrape, crawl, map, search et agent flows
- Un accent fort sur la fiabilite pour les pages JavaScript et le scraping en production
- Un coeur open source avec une offre hebergee pour la mise a l'echelle

## Pourquoi utiliser Firecrawl

- Performant pour le scraping et l'extraction de donnees structurees
- Bien adapte aux sites multi-pages et aux crawls repetes
- Utile quand vous voulez des sorties coherentes

## Connexion via Apps Gallery

1. Ouvrez **Tools Manager**.
2. Allez dans **Apps Gallery**.
3. Recherchez **Firecrawl** et connectez-le.
4. Ajoutez votre cle API et les parametres necessaires.

## Plan gratuit

- Firecrawl propose un plan gratuit.

<Warning>
  Verifiez les limites actuelles de Firecrawl (quotas, rate limits, retention) avant de monter en charge.
</Warning>

## Exemples de prompts

- "Scrape ces 25 pages et extrais titre, auteur, date de publication et sujet principal."
- "Crawl cette section de docs et retourne tous les endpoints avec methode et path."
