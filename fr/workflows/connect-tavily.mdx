Tavily est une bonne option pour la recherche web, l'extraction d'informations et l'exploration d'arborescence.

## Comment Tavily se positionne

- Une couche d'acces web pour les agents IA
- Une API unique pour la recherche en temps reel, l'extraction, la recherche approfondie et le crawling
- Un positionnement centre sur le grounding des modeles avec du contexte web frais et structure
- Un discours enterprise autour de la securite, de la confidentialite et de la fiabilite

## Pourquoi utiliser Tavily

- Bon compromis entre qualite de recherche et profondeur d'extraction
- Utile pour des workflows de type scraping
- Configuration simple via cle API dans **Remote MCP Servers**

## Connexion Tavily

1. Ouvrez **Tools Manager**.
2. Allez dans **Remote MCP Servers**.
3. Ajoutez le serveur Tavily.
4. Collez votre cle API Tavily.
5. Activez le connecteur.

## Plan gratuit

- Tavily est souvent propose avec environ **1 000 credits gratuits par mois**.

<Warning>
  Verifiez la tarification et les limites Tavily avant un usage en production. Les plans peuvent evoluer.
</Warning>
