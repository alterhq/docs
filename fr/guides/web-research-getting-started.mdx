Utilisez ce guide pour choisir la bonne configuration de recherche web dans Alter.

<Info>
  Commencez par les routeurs Alter. `Fast`, `Best`, `Fair` et `Light` ont deja la recherche web activee, sans configuration ni connexion d'outil.
</Info>

## Ce que vous pouvez utiliser

- **Local Tools :** `Web Search` et `Web Search Gemini`
- **Remote MCP Servers :** Tavily, LinkUp, Exa
- **Apps Gallery :** Firecrawl, LinkUp, Exa (si disponibles dans votre workspace)
- **Routeurs Alter :** `Fast`, `Best`, `Fair` et `Light` incluent une recherche web native pour les informations recentes

## Positionnement des providers (apercu)

- **Tavily :** se positionne comme une couche d'acces web pour les agents IA, avec une API unique pour search, extraction, research et crawling.
- **Firecrawl :** se positionne comme une API qui transforme les sites en donnees LLM-ready, avec un focus scraping et crawling.
- **LinkUp :** se positionne comme un moteur de recherche pour AI apps, avec mode standard et mode deep, et des reponses sourcees.
- **Exa :** se positionne comme une API de recherche IA orientee qualite, avec plusieurs endpoints de search a research.

## Construire une strategie hybride

1. Commencez par la recherche native des routeurs, c'est l'option la plus simple avec de bons resultats par defaut.
2. Utilisez les outils de recherche inclus pour etendre les modeles qui n'ont pas de recherche web (y compris les modeles locaux), sans creer de compte hors Alter.
3. Ajoutez des providers MCP quand vous voulez d'excellents resultats sur n'importe quel modele et plus de controle sur la recherche et l'extraction.
4. Ajoutez un scraper si votre workflow exige une extraction structuree multi-pages.

<Warning>
  Les plans gratuits et quotas evoluent. Verifiez la tarification des providers avant de dependre d'une limite precise.
</Warning>

<Tip>
  Evitez d'activer plusieurs outils de recherche web en meme temps. Cela peut perturber le choix d'outil par le modele. Activez un seul outil de recherche par workflow, ou ajoutez des instructions explicites pour imposer un outil precis.
</Tip>
