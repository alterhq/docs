LinkUp est simple a connecter et utile pour la recherche standard comme pour la deep research.

## Comment LinkUp se positionne

- "World's best search for AI apps"
- Un focus sur le grounding factuel avec reponses sourcees et contenus de confiance
- Deux modes clairs : recherche standard rapide et recherche approfondie multi-etapes
- Un positionnement fort pour les usages business (enrichment, answer engines, recherche GTM)

## Options de connexion

- Integration **Apps Gallery**
- **Remote MCP Server** avec cle API

## Connexion LinkUp

1. Ouvrez **Tools Manager**.
2. Connectez LinkUp depuis **Apps Gallery** ou ajoutez-le dans **Remote MCP Servers**.
3. Collez votre cle API.
4. Activez le connecteur.

## Plan gratuit

- LinkUp est souvent presente avec **1 000 requetes standards** et **100 requetes deep research**.

<Warning>
  Confirmez les quotas et conditions LinkUp sur la page de tarification officielle.
</Warning>
