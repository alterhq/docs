Usa esta guía para elegir la mejor configuración de búsqueda web en Alter.

<Info>
  Empieza con los routers de Alter. `Fast`, `Best`, `Fair` y `Light` ya incluyen búsqueda web sin configuración adicional.
</Info>

## Qué puedes usar

- **Local Tools:** `Web Search` y `Web Search Gemini`
- **Remote MCP Servers:** Tavily, LinkUp, Exa
- **Apps Gallery:** Firecrawl, LinkUp, Exa (si están disponibles en tu workspace)
- **Routers Alter:** `Fast`, `Best`, `Fair` y `Light` con búsqueda web nativa

## Resumen del posicionamiento

- **Tavily:** capa de acceso web para agentes de IA, con API para search, extraction, research y crawling.
- **Firecrawl:** API para convertir sitios en datos listos para LLM, con foco en scraping y crawling.
- **LinkUp:** búsqueda para AI apps, con modos estándar y deep y respuestas con fuentes.
- **Exa:** API de búsqueda para IA centrada en calidad, con endpoints de search a research.

## Estrategia híbrida

1. Empieza con búsqueda nativa de routers para tener el setup más fácil y buenos resultados por defecto.
2. Usa tools incluidos para extender modelos sin búsqueda web nativa, incluidos modelos locales, sin crear cuentas fuera de Alter.
3. Añade proveedores MCP cuando quieras grandes resultados en cualquier modelo y más control de investigación y extracción.
4. Añade un scraper cuando necesites extracción estructurada en múltiples páginas.

<Warning>
  Los planes gratuitos y cuotas cambian. Verifica los límites actuales antes de depender de ellos.
</Warning>

<Tip>
  Evita activar varios tools de búsqueda web al mismo tiempo. Puede confundir la selección de tools del modelo.
</Tip>
