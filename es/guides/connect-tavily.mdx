Tavily es una opción fuerte para investigación web, extracción y crawling.

## Cómo se posiciona Tavily

- Capa de acceso web para agentes de IA
- API única para search, extraction, research y crawling
- Enfoque en grounding con contexto web actualizado

## Cómo conectar

1. Abre **Tools Manager**.
2. Entra en **Remote MCP Servers**.
3. Añade Tavily.
4. Pega tu API key.
5. Activa el conector.
