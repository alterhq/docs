### Concepto general

Alter proporciona dos interfaces principales para interactuar con la IA:

<AccordionGroup>
  <Accordion icon="layout-dashboard" title="The Hub">
La interfaz de la ventana principal con pestañas, barra lateral y gestión integral del espacio de trabajo. Lo mejor para:
    - Conversaciones extendidas
    - Workspace gestión
    - Multitarea con múltiples contextos
    - Revisar las transcripciones de las reuniones
  </Accordion>
  
  <Accordion icon="message-circle" title="Panel rápido">
El panel flotante liviano accesible desde la muesca o la tecla de acceso rápido. Perfecto para:
    - consultas rápidas
    - Comandos de voz
    - Adiciones rápidas de contexto
    - Interacciones rápidas
  </Accordion>
</AccordionGroup>

## Interfaz del concentrador

<Frame>
The Hub se divide en tres secciones principales:
  
  1. **Barra lateral**: navegación para conversaciones, espacios de trabajo, transcripciones y más
  2. **Área de contenido principal**: interfaz con pestañas que muestra contenido activo

Abrir un enlace de conversación desde un panel flotante

### Atajos de teclado

| Atajo | Acción |
|----------|--------|
| `Cmd + B` | Alternar visibilidad de la barra lateral |
| `Cmd + W` | Cerrar pestaña activa |
| `Cmd + Option + →` | Pestaña siguiente |
| `Cmd + Option + ←` | Pestaña anterior |
| `Cmd + K` | Borrar todos los contextos |
| `Cmd + Delete` | Eliminar conversación activa |

### Gestión de contextos

<Steps>
  <Step title="Agregar contexto">
Arrastre y suelte archivos, o use `@` en el cuadro de solicitud
  </Step>
  <Step title="Navegar por">
Utilice las teclas de flecha para moverse entre contextos
  </Step>
  <Step title="Claro">
Presione `Cmd + K` para eliminar todos los contextos
  </Step>
</Steps>

  </Card>
  <Card title="3. Selección adicional" icon="plus">
Acceso rápido a modelos y acciones.
  </Card>

### Navegación

Utilice atajos de teclado en el Panel rápido:
- `←`, `↑`, `→`, `↓` - Navegar entre objetos
- `Enter` - Seleccionar un objeto
- `@` - Agregar elementos al contexto
- `#` - Mostrar historial de conversaciones
- `>` - Mostrar historial de reuniones
- `/` - Listar y elegir modelo
- `?` - Mostrar menú de ayuda

<Note>
Consulte la sección **Cuadro de aviso** a continuación para obtener más métodos abreviados de teclado.
</Note>

### Navegación por teclado

| Clave | Acción |
|-----|--------|
| `←` `↑` `→` `↓` | Navegar contextos |
| `Space` | Agregar/eliminar seleccionados |
| `Cmd + K` | Eliminar todos los contextos |

### Usando su portapapeles

Pegar contenido directamente: `Cmd + V`

El contenido de su portapapeles se agrega automáticamente al contexto de Alter.

## El cuadro de aviso

El **Cuadro de aviso** es donde ocurre la magia.

<Frame>
Comience a escribir su mensaje o un título de acción Alter para enviarlo a su modelo de IA.
</Frame>

### Atajos de teclado

| Atajo | Acción |
|----------|--------|
| `Enter` | Enviar mensaje con modelo activo |
| `Cmd + Enter` | Buscar en la web (a través de Gemini) |
| `Shift + Enter` | Nueva línea |
| `@` | Agregar elementos al contexto |
| `#` | Mostrar historial de conversaciones |
| `>` | Mostrar historial de transcripción de reuniones |
| `/` | Listar y elegir modelo |
| `?` | Mostrar menú de ayuda |

<Tip>
**Invocar acciones:** ¡Escribe cualquier carácter en el título de una acción para activarla!
</Tip>

### Habla continua a indicación

Mantenga conversaciones de voz prolongadas sin tocar el teclado:

<Steps>
  <Step title="Mantenga presionada la tecla de acceso rápido de voz">
Mantenga presionado `Fn` (o su clave configurada)
  </Step>
  <Step title="Habla naturalmente">
Di tu mensaje en voz alta
  </Step>
  <Step title="Liberar">
Suelta para enviar a la conversación
  </Step>
  <Step title="Sigue adelante">
Cada mensaje de voz se suma al mismo hilo.
  </Step>
</Steps>

<Tip>
**Flujo de trabajo manos libres:** Perfecto para lluvias de ideas, dictados o cuando tienes las manos ocupadas.
</Tip>

## Historial de conversaciones

### Administrar el historial de chat

Acceda al historial escribiendo `#` en el cuadro de diálogo o usando la barra lateral de The Hub.

<Frame>
![Historial con etiquetas](/images/other/history-tags.webp)
</Frame>

### Organizar con etiquetas

Agregue etiquetas como `#personal` para categorizar los chats:

```
# #personal #food
```

Busque conversaciones que coincidan con varias etiquetas.

### Cambiar el nombre de las conversaciones

<Steps>
  <Step title="Haga clic derecho">
Haga clic derecho en cualquier conversación de la lista
  </Step>
  <Step title="Seleccione Cambiar nombre">
Elija `Rename` del menú
  </Step>
  <Step title="Escribe nuevo nombre">
Dale un nombre descriptivo
  </Step>
</Steps>

### Exportar conversaciones

Exportar como descuento para documentar o compartir:

<Steps>
  <Step title="encontrar conversación">
Localízalo en la historia.
  </Step>
  <Step title="Haga clic derecho">
Abrir el menú contextual
  </Step>
  <Step title="Seleccione Exportar">
Elija la opción Exportar
  </Step>
  <Step title="Ahorrar">
Elija la ubicación y guárdela como archivo .md
  </Step>
</Steps>

<Card title="Usos de exportación" icon="download">
  - Documentación
  - Compartir con el equipo
  - Archivar discusiones importantes
  - Usando en otras Tools
</Card>
