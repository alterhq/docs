## Interactuar con sus aplicaciones

### AppSense - Entendiendo tu Mac

A diferencia de los métodos tradicionales de captura de pantalla, AppSense lee datos reales directamente desde las aplicaciones macOS, lo que permite a Alter acceder a información más allá de lo visible en su pantalla.

<CardGroup cols={2}>
  <Card title="Integración profunda" icon="plug">
Acceda a los datos de la aplicación, no solo a las capturas de pantalla
  </Card>
  <Card title="Conciencia del contexto" icon="brain">
Comprenda en qué está trabajando automáticamente
  </Card>
</CardGroup>

#### Solicitud actual

Alter sugiere automáticamente usar su ventana activa como contexto:

<Steps>
  <Step title="busca la sugerencia">
Haga clic en **Solicitud actual** debajo del cuadro de aviso
  </Step>
  <Step title="O usar el teclado">
Presione `↓` y luego `Enter` para seleccionarlo.
  </Step>
</Steps>

## Follow Mode

Follow Mode es una característica principal de Alter que permite la sincronización en tiempo real de datos de sus aplicaciones, documentos y archivos activos durante las sesiones de chat.

### Cómo funciona

<AccordionGroup>
  <Accordion icon="refresh-cw" title="Monitoreo en tiempo real">
Follow Mode monitorea continuamente el estado de las aplicaciones y archivos con los que está trabajando, brindando a Alter un contexto integral sobre su flujo de trabajo actual en cada paso de su conversación.
  </Accordion>
  
  <Accordion icon="lightbulb" title="Ejemplo práctico">
¿Trabajando en una copia de la página de destino? Mientras realiza cambios y tiene a Alter como su asesor de marketing, Alter ve los cambios que realizó sin necesidad de explicar lo que hizo.
  </Accordion>
</AccordionGroup>

### Usando Follow Mode

<Steps>
  <Step title="Agregar contexto">
Agregue las aplicaciones o documentos que desee incluir a la lista de contexto
  </Step>
  <Step title="Empezar a chatear">
Alter rastrea automáticamente el estado de las aplicaciones seleccionadas
  </Step>
  <Step title="Hacer las cuestiones">
Preguntar sobre el contenido incluso si no está visible actualmente en la pantalla
  </Step>
  <Step title="sigue trabajando">
Interactúa con las aplicaciones seleccionadas o actualiza documentos mientras chateas
  </Step>
</Steps>

<Tip>
**Consejo profesional:** Follow Mode es perfecto para la creación de contenido de formato largo. Agregue su documento al contexto una vez y luego siga escribiendo mientras Alter ​​brinda comentarios en tiempo real.
</Tip>

## Interactuar con archivos y carpetas

### Soporte para arrastrar y soltar

Arrastre y suelte archivos o carpetas en la muesca de The Hub o Alter, o use **Clic derecho > Agregar a Alter**.

### Tipos de archivos admitidos

  <Card title="Imágenes" icon="image">

## Workspaces

Workspaces es una característica poderosa que le permite guardar, organizar y actualizar automáticamente colecciones de carpetas, archivos y aplicaciones para flujos de trabajo eficientes asistidos por IA.

<CardGroup cols={3}>
  <Card title="Respuestas más rápidas" icon="zap">
Recupere solo archivos relevantes para sus consultas
  </Card>
  <Card title="Rentable" icon="credit-card">
Reduzca los costos de procesamiento con indexación inteligente
  </Card>
  <Card title="Maneja grandes proyectos" icon="folder-open">
Trabaje con carpetas grandes y bases de código de manera eficiente
  </Card>
</CardGroup>

### Cómo funciona Workspaces

Workspaces crea un mapa indexado de sus recursos seleccionados. Los archivos se convierten automáticamente:

| Tipo de archivo | Conversión |
|-----------|-----------|
| **PDF** | Markdown con marcadores de página |
| **DOCX** | formato Markdown |
| **XLSX** | Markdown/formato CSV |
| **Imágenes** | OCR extracción de texto |
| **Directorios** | Procesamiento recursivo |

<Note>
Los espacios de trabajo aún no admiten archivos de audio y vídeo. Transcríbalos primero ingresando a Alter.
</Note>

### Edición de archivos en Workspaces

<Tabs>
  <Tab title="Edición única">
Edite o cree archivos individuales usando lenguaje natural:
    - Modificar código
    - Actualizar documentación
    - Funciones de refactorización
    - Crea nuevos archivos desde cero
  </Tab>
  
  <Tab title="Edición múltiple">
Edite varios archivos simultáneamente:
    - Perfecto para refactorizar
    - Actualizar importaciones entre archivos
    - Cambios consistentes en toda la base de código
  </Tab>
  
  <Tab title="Golpe Tool">
Ejecute comandos bash para operaciones avanzadas:
    - Ejecutar scripts
    - Operaciones por lotes
    - Automatización de flujos de trabajo complejos
  </Tab>
</Tabs>

<Info>
**La seguridad es lo primero:** Todas las operaciones de edición están protegidas mediante políticas de cinturones de seguridad con controles de seguridad para una ejecución segura.
</Info>

### Creando un Workspace

<Steps>
  <Step title="Abrir The Hub">
Presione `Shift + Cmd + H` o seleccione desde la barra de menú
  </Step>
  <Step title="Vaya a Workspaces">
Busque la sección Workspaces en la barra lateral
  </Step>
  <Step title="Crear nuevo">
Haga clic en el botón **+** en el selector de espacio de trabajo.
  </Step>
  <Step title="Agregar archivos">
Utilice el botón "Agregar archivos" para seleccionar carpetas y archivos
  </Step>
</Steps>

### Workspace Almacenamiento

Workspaces se almacenan localmente en:
```
~/Library/Application Support/Alter/Workspaces/
```

Cada espacio de trabajo contiene:
- Workspace definición (`.alterworkspace`)
- Tus archivos y carpetas
- Archivos convertidos en caché para un acceso rápido

## Alter Acciones - Flujos de trabajo

Alter Las acciones son indicaciones avanzadas configuradas con activadores e instrucciones.

### Creando acciones

<CardGroup cols={2}>
  <Card title="Editor de acciones" icon="edit">
Accesible desde la barra de menú para un control total
  </Card>
  <Card title="Creación rápida" icon="plus">
Escriba "Crear > Alter Acción" en el cuadro de diálogo.
  </Card>
</CardGroup>

<Note>
Las acciones integradas no se pueden editar, pero se pueden duplicar y personalizar.
</Note>

### Selección de modelos a medida

Asigne modelos de IA específicos a acciones individuales para un control preciso.

#### ¿Por qué utilizar diferentes modelos?

<AccordionGroup>
  <Accordion icon="shield" title="Flujos de trabajo que priorizan la privacidad">
Utilice modelos locales para tareas confidenciales y mantenga los modelos en la nube para consultas generales.
  </Accordion>
  
  <Accordion icon="gauge" title="Rendimiento especializado">
Asigne modelos rápidos y optimizados para Tools, como Qwen3, para la automatización mientras utiliza modelos creativos para escribir.
  </Accordion>
  
  <Accordion icon="piggy-bank" title="Optimización de costos">
Utilice modelos locales livianos para tareas simples y reserve modelos potentes para operaciones complejas
  </Accordion>
</AccordionGroup>

#### Configuración

<Steps>
  <Step title="Abrir editor de acciones">
Haga clic en los tres puntos en la ventana Alter
  </Step>
  <Step title="Seleccionar acción">
Elija la acción Alter para personalizar
  </Step>
  <Step title="Ir a avanzado">
Busque la sección "Avanzado"
  </Step>
  <Step title="Elige modelo">
Selecciona el modelo específico para esta acción
  </Step>
  <Step title="Ahorrar">
La acción ahora usa el modelo elegido.
  </Step>
</Steps>

### Personalización del nivel de acción Tool

Elija exactamente qué Tools puede utilizar cada acción.

#### Beneficios

<CardGroup cols={2}>
  <Card title="Ejecución más rápida" icon="zap">
Limite las Tools a solo lo necesario
  </Card>
  <Card title="Mejor precisión" icon="target">
Evite que la IA elija Tools irrelevantes
  </Card>
  <Card title="Consciente del contexto" icon="brain">
Crea flujos de trabajo especializados por acción
  </Card>
  <Card title="Rentable" icon="credit-card">
Reducir el uso de tokens
  </Card>
</CardGroup>

#### Configuración

<Steps>
  <Step title="Abrir editor de acciones">
Haga clic en tres puntos → "Editor de acciones"
  </Step>
  <Step title="Seleccionar acción">
Elija la acción Alter
  </Step>
  <Step title="Busque la sección Tools">
Busque en la configuración "Avanzada"
  </Step>
  <Step title="Elige Tools">
Seleccione Tools específicas o deshabilite todas
  </Step>
</Steps>

#### Tool Establecer estrategias

- **Mínimo:** Una o dos Tools para acciones enfocadas
- **Agrupados:** Herramientas relacionadas para flujos de trabajo específicos
- **Acceso completo:** Todas las Tools para acciones de propósito general
- **No Tools:** Para tareas creativas o analíticas

<Tip>
**Consejo profesional:** Comience con todas las Tools deshabilitadas para nuevas acciones, luego agregue solo lo que necesite. Esto garantiza un rendimiento óptimo y evita el uso inesperado de Tools.
</Tip>
