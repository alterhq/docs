---

Alter incluye dos tools de búsqueda web listos para usar.

Estos tools ayudan cuando usas modelos sin búsqueda web nativa, incluidos modelos locales.

<Info>
  A diferencia de Tavily, LinkUp, Firecrawl y Exa, estos tools están incluidos con el uso de Alter y no requieren cuenta de proveedor externa.
</Info>

## Comparación

<CardGroup cols={2}>
  <Card title="Web Search (DuckDuckGo)" icon="compass">
    Sin API key. Gratis y disponible al instante. Usa snippets, así que es rápido pero menos detallado.
  </Card>
  <Card title="Web Search Gemini" icon="sparkles">
    Usa Google search con Gemini 2.5 Flash. Más detalle y enlaces de fuentes. Requiere Alter Pro o Lifetime.
  </Card>
</CardGroup>

## Cuándo usar cada uno

- Usa **Web Search (DuckDuckGo)** con cuenta gratuita o para búsquedas rápidas.
- Usa **Web Search Gemini** con Pro/Lifetime para más profundidad y fuentes.

## Cómo activar

1. Abre **Tools Manager**.
2. Ve a **Local Tools**.
3. Activa **Web Search** y/o **Web Search Gemini**.

<Tip>
  No dejes dos tools de búsqueda activos por defecto. Activa uno por tarea o añade una instrucción explícita sobre qué tool usar.
</Tip>
