Exa es un provider centrado en búsqueda web para IA.

## Cómo se posiciona Exa

- "The best search API for AI"
- Foco en calidad y benchmarks
- Endpoints para search, contents, answer, crawl y research

## Cómo conectar

1. Abre **Tools Manager**.
2. Añade Exa en **Remote MCP Servers** o en **Apps Gallery**.
3. Pega tu API key.
4. Activa el conector.
