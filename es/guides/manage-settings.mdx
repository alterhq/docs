## Meta

Configure Alter valores predeterminados, apariencia, dictado, permisos y perfil.

## Pasos

1. Abra Alter, luego presione `Cmd+,`.
2. Configure **Valores predeterminados** para un comportamiento de acceso rápido.
3. Configure **Apariencia** para la visibilidad y el tema de la muesca.
4. Configure **Dictado** para procesador, idioma y reemplazos.
5. Configure **Permisos** y otorgue el acceso requerido a macOS.
6. Configure **Perfil** para función, proyectos y estilo de comunicación.
