LinkUp es fácil de conectar para búsqueda estándar y deep research.

## Cómo se posiciona LinkUp

- "World's best search for AI apps"
- Respuestas con fuentes y foco en grounding factual
- Modos Standard y Deep

## Cómo conectar

1. Abre **Tools Manager**.
2. Conecta desde **Apps Gallery** o **Remote MCP Servers**.
3. Pega tu API key.
4. Activa el conector.
