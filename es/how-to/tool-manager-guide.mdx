## Meta

Conecte y administre integraciones de forma segura con pasos claros para la solución de problemas.

## Pasos

1. Abra la configuración de Alter y vaya al administrador de integraciones o Tools.
2. Conecte una integración a la vez y pruébela inmediatamente.
3. Confirme los permisos requeridos y los alcances de la cuenta.
4. Cree un mensaje de prueba simple para verificar el comportamiento esperado.
5. Si se producen fallas, revise la autenticación del proveedor y vuelva a intentarlo.

## Administrar muchos Tools

Habilitar muchas Tools aumenta el uso de la ventana de contexto y afecta el rendimiento. Para la mayoría de los flujos de trabajo, utilice **Flow**: el orquestador de Tools de Alter que selecciona automáticamente la Tool adecuada para cada solicitud.

<Card title="Utilice Flow para la orquestación Tool" icon="wand-magic-sparkles" href="/workflows/use-flow-orchestration">
Flow reduce el uso del contexto al actuar como una única Tool que enruta las solicitudes a la integración adecuada automáticamente.
</Card>

En lugar de cargar todas las Tools en cada solicitud, Flow:
- Analiza su solicitud y selecciona la Tool adecuada
- Maneja la orquestación de varios pasos en todas las Tools.
- Reduce el uso de la ventana de contexto y mejora el rendimiento.

Habilite Flow en Tool Manager en **Local Tools > Alter**.

## Solución de problemas

- Vuelva a autenticar la conexión del proveedor.
- Confirme los permisos locales/de red.
- Verifique los límites y cuotas específicos de la integración.
