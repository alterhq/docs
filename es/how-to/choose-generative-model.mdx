De forma predeterminada, Alter ofrece una variedad de modelos, enumerados alfabéticamente por empresa de alojamiento.

También puede usar su [clave API](#via-your-api-key) o usar [su modelo local](#locally-via-ollama-or-studiolm).

### Eligiendo tu modelo

Después de abrir Alter, simplemente escriba `/` para mostrar los modelos disponibles y seleccione el modelo que desea usar.


#### Elegir un modelo favorito

Seleccione el ícono de estrella al lado de un modelo con el mouse para marcar el modelo como su favorito.
![Favoreciendo un modelo](/images/models/favorite-model.webp)


Cuando utiliza su clave API o un modelo local, estos aparecen en la parte superior en **Personalizado**.
Todas las demás interacciones se enrutan a través de Alter Cloud.

### Usando Alter Nube

De forma predeterminada, Alter está configurado con `/best`, nuestro modelo de enrutador interno, eligiendo el mejor modelo según su consulta.

![Ejemplo de enrutador y modelo](/images/settings/router.png)


Puede cambiar el modelo predeterminado en **Configuración > Valores predeterminados > Modelo**.

**Para obtener la lista actual completa de los más de 92 modelos**, consulte los modelos disponibles directamente a través de la API:

```bash
curl https://alterhq.com/api/models \
  -H "Authorization: Bearer YOUR_API_KEY"
```

O consulte la **[API Gateway Guide](/api-router/api-gateway)** para obtener información detallada sobre la selección y el uso del modelo.

### Usando su clave API

![Personalizado Local](/images/models/custom-api-keys.webp)

Puede utilizar su propia clave API para conectarse directamente al punto final de su proveedor:
1. Vaya a **Configuración > Claves API > Proveedor personalizado**.
2. Habilitar proveedor personalizado
3. Elija su proveedor en la lista

> Una vez conectado, aparecerá una marca junto a **Punto final personalizado**.

Los modelos de su proveedor aparecerán en la sección **Personalizado**.

> Solo admitimos puntos finales compatibles con OpenAI, lo que significa que no puede usar sus claves API de Anthropic.

A continuación se muestra una lista de las plataformas compatibles con claves personalizadas, según la imagen proporcionada:

1.  Azur
2.  Gemini
3.  Estudio LM
4.  Mistral
5.  Ollama
6.  OpenAI
7.  Enrutador abierto
8.  Personalizado (cualquier terminal compatible con OpenAI)

#### Configuración de un punto final de Azure en Alter

1.  **Habilitar proveedor personalizado:** En la configuración de Alter, habilite la opción "Proveedor personalizado".
2.  **Elija Azure:** Seleccione Azure como su proveedor.
3.  **Ingrese la URL completa del punto final de finalización del chat:** Proporcione la URL completa del punto final de finalización del chat, incluido el parámetro `api-version` y el nombre del modelo dentro de la URL.  Por ejemplo: `https://yourinstance.openai.azure.com/openai/deployments/your-model-name/chat/completions?api-version=2024-02-15-preview`
4.  **Ingrese la clave API:** Ingrese su clave API OpenAI de Azure.

Consideraciones importantes

*   **Se requiere un punto final completo:** Asegúrese de utilizar el punto final de finalización de chat *completo*, ya que Alter extrae el nombre del modelo de la URL.
*   **Nombre del modelo en la URL:** El nombre de implementación del modelo *debe* incluirse en la ruta URL.
*   **Versión API:** La URL *debe* incluir el parámetro `api-version`.


#### Dónde encontrar sus claves API

| Proveedor | URL para generar una clave API |
| -------- | -------- | 
| Google (Gemini) | https://aistudio.google.com/app/apikey | [Enlace](https://ai.google.dev/api/compatibility)
| Mistral | https://console.mistral.ai/api-keys/
| OpenAI | https://platform.openai.com/api-keys |[Enlace]




### Localmente usando Ollama o LM Studio

![Personalizado Local](https://beehiiv-images-production.s3.amazonaws.com/uploads/asset/file/a3cb944c-7379-49a6-9349-c829bcda071b/image.png?t=1740499713)

Para usar Alter con modelos locales (a través de Ollama o LM Studio):
1. Vaya a **Configuración > Claves API > Proveedor personalizado**.
2. Habilitar proveedor personalizado
3. Elija Ollama o LM Studio en la lista de proveedores

> Una vez conectado, aparecerá una marca junto a **Punto final personalizado**.

![imagen alt](https://beehiiv-images-production.s3.amazonaws.com/uploads/asset/file/4a22f3fb-ad9e-4f50-8184-1e5cc5238118/Alter_Settings_API_key_custom_provider_on.png?t=1740500360)

Sus modelos locales aparecerán en la sección **Personalizado**.

![Modelo personalizado](https://beehiiv-images-production.s3.amazonaws.com/uploads/asset/file/c3a9bef5-ff23-4a35-99b6-aedc7bc38972/Alter_Main_Custom_Models.png?t=1740502675)
