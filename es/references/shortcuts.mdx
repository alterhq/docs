### Navegación global

Disponible en todas las ventanas (a menos que la ventana de destino ya esté activa).

- **Mayús + Cmd + H**: Abrir Alter Hub
- **Shift + Cmd + E**: Abrir editor de acciones
- **Mayús + Cmd + T**: Abrir Tools Manager
- **Alt + Espacio** (predeterminado): Muesca abierta: configurable en **Configuración > Valores predeterminados > Combinación de teclas de muesca abierta**
- **Alter Acceso directo al menú rápido**: abra Alter desde cualquier lugar; se puede configurar en **Configuración > General > Acceso rápido**
- **Tecla de acceso rápido de voz a texto**: inicia la entrada de voz; se puede configurar en **Configuración > Dictado > Acceso directo** (predeterminado: tecla Fn)

### Panel rápido

Cuando el panel rápido está abierto (desde la muesca):

- **←**, ** ↑**, **→**, **↓**: Navegar entre objetos
- **Entrar**: Selecciona un objeto

### Sección de contexto

- **←**, ** ↑**, **→**, **↓**: Navegar entre contextos
- **Barra espaciadora**: Agregar/eliminar contexto(s) seleccionado(s)
- **Cmd + K**: Elimina todos los contextos de la conversación activa
- **Cmd + V**: Pega el contenido del portapapeles para agregarlo al contexto

### Cuadro de aviso

- **Entrar**: Pregunte con su modelo activo
- **Cmd + Enter**: Solicitar mediante la búsqueda web (a través de Gemini web)
- **Mayús + Intro**: salto de línea
- **@**: Agregar elementos al contexto
- **#**: Mostrar historial de Alter conversaciones
- **>**: muestra el historial de transcripciones de reuniones
- **/**: Lista y elige tu modelo
- **?**: Mostrar menú de ayuda

### Alter Ventana central

- **Cmd + F**: Búsqueda de enfoque (teclas de flecha para navegar por los resultados)
- **Cmd + B**: alternar visibilidad de la barra lateral
- **Cmd + T**: Nueva pestaña (iniciar nuevo chat)
- **Cmd + J**: Crear espacio de trabajo (requiere contexto activo)
- **Cmd + Opc + O**: Cambiar al espacio de trabajo...
- **Cmd + K**: Borrar todos los contextos (requiere contextos activos)
- **Cmd + Eliminar**: Eliminar conversación (requiere conversación activa)
- **Cmd + W**: Cerrar pestaña (cierra la ventana si es la última pestaña)
- **Cmd + Opc + W**: cerrar todas las pestañas (requiere >1 pestaña)
- **Cmd + Opc + ←**: Pestaña anterior (requiere >1 pestaña)
- **Cmd + Opc + →**: Pestaña siguiente (requiere >1 pestaña)
- **Cmd + 1...9**: Cambiar al índice de pestaña específico

#### Workspace (requiere espacio de trabajo seleccionado)

No hay atajos predeterminados; Estos se pueden configurar a través de la configuración del sistema macOS:

- Abrir configuración del espacio de trabajo
- Cambiar nombre del espacio de trabajo
- Establecer permisos
- Alternar zona de pruebas
- Iniciar/detener contenedor de espacio aislado
- Abrir terminal en sandbox

### Ventana del editor de acciones

- **Cmd + F**: Búsqueda de enfoque
- **Cmd + N**: Nueva acción
- **Cmd + D**: acción duplicada (requiere acción seleccionada)
- **Cmd + E**: Exportar acción (requiere acción seleccionada)
- **Cmd + O**: Importar acción
- **Cmd + Eliminar**: Eliminar acción (requiere acción seleccionada)
- **Cmd + W**: Cerrar ventana
- **Cmd + [**: Sección anterior
- **Cmd + ]**: Siguiente sección

### Tools Manager Ventana

- **Cmd + W**: Cerrar ventana
- **Cmd + F**: Búsqueda de enfoque
- **Cuando se selecciona "Habilidades"**: Cmd + O (Importar habilidad), Cmd + R (Actualizar lista)
- **Cuando se selecciona "Servidores MCP remotos"**: Cmd + N (Agregar servidor), Cmd + R (Actualizar lista)
- **Cuando se selecciona "Acciones rápidas"**: Cmd + N (Nueva acción rápida), Eliminar (Eliminar acción rápida, requiere selección)

### Paneles de reuniones en vivo

Disponible solo durante las reuniones:

- **Cmd + Shift + ↓**: activar/desactivar los subtítulos en vivo
- **Shift + Cmd + →**: alternar el bloc de notas en vivo

### Alter Configuración

- **Cmd +,**: Abrir configuración (cuando la ventana Alter está visible)
