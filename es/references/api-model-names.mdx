Esta referencia enumera todos los modelos de IA disponibles a través de Alter API Router. Utilice estos nombres de modelo al realizar solicitudes de API.

## Formato de denominación del modelo

Todos los modelos siguen el formato: `Provider#Model-name`

Por ejemplo: `OpenAI#gpt-5`, `Claude#claude-sonnet-4-6`

## Ejemplos de uso

### Pitón
```python
from openai import OpenAI

client = OpenAI(
    api_key="YOUR_API_KEY",
    base_url="https://alterhq.com/api/v1"
)

# Use GPT-5
response = client.chat.completions.create(
    model="OpenAI#gpt-5",
    messages=[{"role": "user", "content": "Hello!"}]
)
```

### rizo
```bash
curl https://alterhq.com/api/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "model": "OpenAI#gpt-5",
    "messages": [{"role": "user", "content": "Hello!"}]
  }'
```
