## ¿Cómo abro Alter?

1. Pase el cursor sobre la muesca.
2. Presione `Alt+Space`.
3. Mantenga presionado `Fn` y hable.

![Abrir Alter](/images/getting-started/open-notch.gif)

Consulte [Cómo comenzar](/guides/getting-started)] para obtener más detalles.

## ¿Cómo agrego mi propia clave API?

1. Abra Configuración (`Cmd+,`).
2. Vaya a Claves API.
3. Seleccione proveedor e ingrese la clave.

![Claves API personalizadas](/images/models/custom-api-keys.webp)

Consulte [Cómo hacerlo: use su propia clave API](/how-to/use-byok) para obtener instrucciones detalladas.

## ¿Cómo agrego contexto a un mensaje?

- Arrastre y suelte archivos en la muesca.
- Haga clic derecho en los archivos y elija Agregar a Alter.
- Utilice `@` para agregar contexto desde el mensaje.

![Arrastrar y soltar](/images/context/drag-and-drop.gif)

Consulte [La ventana Alter](/guides/alter-window#the-context-section) para obtener más detalles.

## ¿Cómo cambio la configuración rápidamente?

- Abra Alter y presione `Cmd+,`.
- O abra el menú de tres puntos -> Configuración.

![Menú de configuración](/images/settings/settings-menu.png)

Consulte [Alter Configuración](/guides/alter-settings) para obtener más detalles.

## ¿Cómo configuro el dictado?

Configuración -> Dictado, luego elija procesador, idioma y reemplazos.

![Dictado](/images/settings/dictation.webp)

Consulte la [Guía de dictado](/guides/dictation) para obtener instrucciones de configuración completas.
