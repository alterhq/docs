## Local Tools

Alter proporciona **más de 17 potentes Tools locales** que se integran directamente con sus aplicaciones Mac. A diferencia de otros asistentes de IA, Alter en realidad controla tu Mac a través del lenguaje natural.

<CardGroup cols={2}>
  <Card title="100% gratis" icon="gift">

  </Card>
  <Card title="Funciona sin conexión" icon="wifi-off">
No se requiere Internet: completamente privado
  </Card>
  <Card title="Control nativo" icon="command">
Controla realmente tus aplicaciones, no solo lee sobre ellas
  </Card>
  <Card title="Privacidad primero" icon="shield">
Tus datos nunca salen de tu Mac
  </Card>
</CardGroup>

### Cómo utilizar Local Tools

<Steps>
  <Step title="Abierto Tools Manager">
Haga clic en los tres puntos en Alter → **Tools Manager**
  </Step>
  <Step title="Habilitar Tools">
En la sección **Local Tools**, busque y habilite Tools (Calendario, Mapas, etc.)
  </Step>
  <Step title="Usa lenguaje natural">
Solicite a Alter que realice acciones:
    - "Envía un mensaje a John diciéndole que llego tarde"
    - "Agregar una reunión a mi calendario para mañana a las 2 p.m."
    - "Muéstrame mis correos electrónicos no leídos"
  </Step>
</Steps>

<Note>
Active/desactive todas las Tools usando el interruptor global en Tools Manager (Configuración > Tools)
</Note>

## x-callback-url Integración

Alter admite el protocolo [x-callback-url](https://x-callback-url.com/) para la comunicación entre aplicaciones.

### ¿Qué es x-callback-url?

Un estándar que permite que las aplicaciones activen acciones entre sí mediante URL especiales. Pase datos, realice acciones y regrese sin problemas.

### Activación de Alter desde otras aplicaciones

<Steps>
  <Step title="Obtener la URL">
En Action Editor, seleccione una acción → sección "URL Callback"
  </Step>
  <Step title="Copiar esquema">
Copie la URL `alter://` para esa acción.
  </Step>
  <Step title="Usar en cualquier lugar">
Pegar en Atajos, Notas, navegador, etc.
  </Step>
</Steps>

**Ejemplo:**
```
alter://action/ask-web?input=What+is+Alter+MacOS
```

### Activar otras aplicaciones desde Alter

Cree enlaces en los que se pueda hacer clic utilizando esquemas de URL de la aplicación de destino:

| Aplicación | URL de ejemplo | Acción |
|-----|-------------|--------|
| Oso | `bear://x-callback-url/create?title=Demo` | Crear nota |
| Cosas | `things://add?title=Task` | Agregar tarea |
| Borradores | `drafts://x-callback-url/create?text=Hello` | Crear borrador |

<Card title="Aplicaciones compatibles" icon="check-circle">
Bear, Things, Drafts, OmniFocus, 1Writer, Agenda y cientos más
</Card>

<Info>
Consulte la guía [URL Callbacks](/workflows/url-callbacks) para obtener la documentación completa
</Info>

## Solución de problemas

### Problemas comunes

<AccordionGroup>
  <Accordion icon="x-circle" title="Tool No funciona">
Compruebe qué parámetros se requieren. Cree indicaciones personalizadas o agregue explicaciones a su perfil para un mejor uso de la Tool.
  </Accordion>
  
  <Accordion icon="cpu" title="Asistente no utiliza Tools">
Asegúrese de que su modelo admita Tools. Recomendado:
    - Alter Mejor
    - Gemini 2.0 Flash
    - Claude Soneto

  <Accordion icon="shortcut" title="Problemas con los atajos de Apple">

  </Accordion>
