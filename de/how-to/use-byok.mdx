## Schritte

1. Öffnen Sie die Einstellungen mit `Cmd+,`.
2. Öffnen Sie die API-Schlüssel.
3. Wählen Sie den Anbieter und fügen Sie den Schlüssel ein.
4. Geben Sie `/` im Prompt ein, um zu bestätigen, dass benutzerdefinierte Modelle angezeigt werden.
