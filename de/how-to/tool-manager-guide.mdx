## Ziel

Sichere Verbindung und Verwaltung von Integrationen mit klaren Fehlersuche-Schritten.

## Schritte

1. Öffnen Sie die Alter-Einstellungen und gehen Sie zu Integrationen oder Tool-Manager.
2. Verbinden Sie jeweils eine Integration und testen Sie sofort.
3. Bestätigen Sie die erforderlichen Berechtigungen und Kontobereich.
4. Erstellen Sie ein einfaches Testprompt, um das erwartete Verhalten zu überprüfen.
5. Wenn Fehler auftreten, überprüfen Sie die Authentifizierung des Anbieters und versuchen Sie es erneut.

## Verwaltung vieler Tools

Die Aktivierung vieler Tools erhöht die Nutzung des Kontextfensters und wirkt sich auf die Leistung aus. Für die meisten Workflows verwenden Sie **Flow** - Alters Tool-Orchestrator, der automatisch das richtige Tool für jede Anfrage auswählt.

<Card title="Flow für die Tool-Orchestrierung verwenden" icon="wand-magic-sparkles" href="/de/workflows/use-flow-orchestration">
  Flow reduziert die Kontextnutzung, indem es als einzelnes Tool fungiert, das Anfragen automatisch an die entsprechende Integration weiterleitet.
</Card>

Anstatt alle Tools in jede Anfrage zu laden, macht Flow:
- Analysiert Ihre Anfrage und wählt das richtige Tool aus
- Behandelt die mehrstufige Orchestrierung über Tools hinweg
- Reduziert die Nutzung des Kontextfensters und verbessert die Leistung

Aktivieren Sie Flow im Tool Manager unter **Local Tools > Alter**.

## Fehlersuche

- Authentifizieren Sie die Anbieterverbindung erneut.
- Bestätigen Sie die lokalen/Netzwerkberechtigungen.
- Überprüfen Sie die spezifischen Grenzen und Quoten der Integration.
