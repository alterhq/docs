Tavily ist eine starke Option fuer Webrecherche, Extraktion und crawling.

## So positioniert sich Tavily

- Webzugriffsschicht fuer KI-Agenten
- Eine API fuer search, extraction, research und crawling
- Fokus auf grounding mit aktuellem Webkontext

## Verbindung einrichten

1. Oeffnen Sie **Tools Manager**.
2. Gehen Sie zu **Remote MCP Servers**.
3. Fuegen Sie Tavily hinzu.
4. Fuegen Sie den API-Key ein.
5. Aktivieren Sie den Konnektor.
