title: "Enthaltene Such-Tools"
description: "Nutzen Sie die in Alter enthaltenen Websuche-Tools ohne externes Konto"
---

Alter enthaelt zwei integrierte Websuche-Tools.

Diese Tools sind hilfreich, wenn Sie Modelle ohne native Websuche nutzen, inklusive lokaler Modelle.

<Info>
  Im Unterschied zu Tavily, LinkUp, Firecrawl oder Exa sind diese Tools in Alter enthalten und benoetigen kein zusaetzliches Provider-Konto.
</Info>

## Vergleich

<CardGroup cols={2}>
  <Card title="Web Search (DuckDuckGo)" icon="compass">
    Kein API-Key. Kostenlos und sofort verfuegbar. Nutzt Snippets, daher schnell, aber weniger detailliert.
  </Card>
  <Card title="Web Search Gemini" icon="sparkles">
    Nutzt Google search mit Gemini 2.5 Flash. Detailliertere Antworten mit Quellen. Benoetigt Alter Pro oder Lifetime.
  </Card>
</CardGroup>

## Wann welches Tool

- Nutzen Sie **Web Search (DuckDuckGo)** mit kostenlosem Konto oder fuer schnelle Checks.
- Nutzen Sie **Web Search Gemini** mit Pro/Lifetime fuer tiefere Antworten mit Quellen.

## Aktivierung

1. Oeffnen Sie **Tools Manager**.
2. Gehen Sie zu **Local Tools**.
3. Aktivieren Sie **Web Search** und/oder **Web Search Gemini**.

<Tip>
  Lassen Sie nicht beide Such-Tools standardmaessig aktiv. Aktivieren Sie ein Tool pro Aufgabe oder geben Sie eine klare Tool-Anweisung.
</Tip>
