Exa ist ein Provider mit Fokus auf KI-Websuche.

## So positioniert sich Exa

- "The best search API for AI"
- Fokus auf Qualitaet und Benchmarks
- Endpoints fuer search, contents, answer, crawl und research

## Verbindung einrichten

1. Oeffnen Sie **Tools Manager**.
2. Fuegen Sie Exa in **Remote MCP Servers** hinzu oder verbinden Sie es ueber **Apps Gallery**.
3. Fuegen Sie den API-Key ein.
4. Aktivieren Sie den Konnektor.

## Free Tier

- Wird oft mit etwa **1.000 Requests pro Monat** beschrieben.

<Warning>
  Verifizieren Sie Preise und Limits waehrend der Einrichtung.
</Warning>
