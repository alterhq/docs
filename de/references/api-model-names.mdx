Diese Referenz listet alle KI-Modelle auf, die über den Alter API Router verfügbar sind. Verwenden Sie diese Modellnamen, wenn Sie API-Anfragen stellen.

## Modellbenennungsformat

Alle Modelle folgen dem Format: `Provider#Modellname`

Zum Beispiel: `OpenAI#gpt-5`, `Claude#claude-sonnet-4-6`

## Anwendungsbeispiele

### Python
```python
from openai import OpenAI

client = OpenAI(
    api_key="YOUR_API_KEY",
    base_url="https://alterhq.com/api/v1"
)

# Verwenden Sie GPT-5
response = client.chat.completions.create(
    model="OpenAI#gpt-5",
    messages=[{"role": "user", "content": "Hallo!"}]
)
```

### cURL
```bash
curl https://alterhq.com/api/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "model": "OpenAI#gpt-5",
    "messages": [{"role": "user", "content": "Hallo!"}]
  }'
```
