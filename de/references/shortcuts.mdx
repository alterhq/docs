### Globale Navigation

Verfügbar in allen Fenstern (es sei denn, das Ziel-Fenster ist bereits aktiv).

- **Shift + Cmd + H**: Alter Hub öffnen
- **Shift + Cmd + E**: Aktionseditor öffnen
- **Shift + Cmd + T**: Tools Manager öffnen
- **Alt + Space** (Standard): Notch öffnen — konfigurierbar in **Einstellungen > Standard > Notch-Tastenkombination**
- **Alter Quick Menu-Tastenkombination**: Alter von überall öffnen — konfigurierbar in **Einstellungen > Allgemein > Schnellzugriff**
- **Spracherkennung-Hotkey**: Sprachinput starten — konfigurierbar in **Einstellungen > Diktat > Tastenkombination** (Standard: Fn-Taste)

### Schnellpanel

Wenn das Schnellpanel geöffnet ist (von der Notch):

- **←**, **↑**, **→**, **↓**: Zwischen Objekten navigieren
- **Enter**: Ein Objekt auswählen

### Kontextbereich

- **←**, **↑**, **→**, **↓**: Zwischen Kontexten navigieren
- **Leertaste**: Ausgewählte Kontext(e) hinzufügen/entfernen
- **Cmd + K**: Alle Kontexte aus dem aktiven Gespräch entfernen
- **Cmd + V**: Inhalt aus der Zwischenablage einfügen, um ihn zum Kontext hinzuzufügen

### Eingabeaufforderungsfeld

- **Enter**: Eingabeaufforderung mit deinem aktiven Modell
- **Cmd + Enter**: Eingabeaufforderung mit Websuche (über Gemini-Web)
- **Shift + Enter**: Zeilenumbruch
- **@**: Elemente zum Kontext hinzufügen
- **#**: Verlauf der Alter-Gespräche anzeigen
- **>**: Verlauf der Meeting-Transkripte anzeigen
- **/**: Dein Modell auflisten und auswählen
- **?**: Hilfemenü anzeigen

### Alter Hub-Fenster

- **Cmd + F**: Suche fokussieren (Pfeiltasten zur Navigation in den Ergebnissen)
- **Cmd + B**: Sichtbarkeit der Seitenleiste umschalten
- **Cmd + T**: Neuer Tab (neuen Chat starten)
- **Cmd + J**: Workspace erstellen (erfordert aktiven Kontext)
- **Cmd + Opt + O**: Zu Workspace wechseln...
- **Cmd + K**: Alle Kontexte löschen (erfordert aktive Kontexte)
- **Cmd + Delete**: Gespräch löschen (erfordert aktives Gespräch)
- **Cmd + W**: Tab schließen (schließt das Fenster, wenn es der letzte Tab ist)
- **Cmd + Opt + W**: Alle Tabs schließen (erfordert >1 Tab)
- **Cmd + Opt + ←**: Vorheriger Tab (erfordert >1 Tab)
- **Cmd + Opt + →**: Nächster Tab (erfordert >1 Tab)
- **Cmd + 1...9**: Zu spezifischem Tab-Index wechseln

#### Workspace (erfordert ausgewählten Workspace)

Keine Standard-Tastenkombinationen; diese können in den macOS Systemeinstellungen festgelegt werden:

- Workspace-Einstellungen öffnen
- Workspace umbenennen
- Berechtigungen festlegen
- Sandbox umschalten
- Sandbox-Container starten/stoppen
- Terminal in der Sandbox öffnen

### Aktionseditor-Fenster

- **Cmd + F**: Suche fokussieren
- **Cmd + N**: Neue Aktion
- **Cmd + D**: Aktion duplizieren (erfordert ausgewählte Aktion)
- **Cmd + E**: Aktion exportieren (erfordert ausgewählte Aktion)
- **Cmd + O**: Aktion importieren
- **Cmd + Delete**: Aktion löschen (erfordert ausgewählte Aktion)
- **Cmd + W**: Fenster schließen
- **Cmd + [**: Vorheriger Abschnitt
- **Cmd + ]**: Nächster Abschnitt

### Tools Manager-Fenster

- **Cmd + W**: Fenster schließen
- **Cmd + F**: Suche fokussieren
- **Wenn "Skills" ausgewählt ist**: Cmd + O (Skill importieren), Cmd + R (Liste aktualisieren)
- **Wenn "Remote MCP Servers" ausgewählt ist**: Cmd + N (Server hinzufügen), Cmd + R (Liste aktualisieren)
- **Wenn "Quick Actions" ausgewählt ist**: Cmd + N (Neue Schnellaktion), Delete (Schnellaktion löschen, Auswahl erforderlich)

### Live-Meeting-Panels

Nur während Meetings verfügbar:

- **Cmd + Shift + ↓**: Live-Untertitel ein/ausschalten
- **Shift + Cmd + →**: Live-Notizbuch umschalten

### Alter-Einstellungen

- **Cmd + ,**: Einstellungen öffnen (wenn das Alter-Fenster sichtbar ist)
