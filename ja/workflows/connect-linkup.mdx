LinkUp は standard search と deep research の両方に使いやすい接続です。

## LinkUp の位置付け

- "World's best search for AI apps"
- ソース付き回答による factual grounding
- Standard と Deep の 2 モード

## 接続手順

1. **Tools Manager** を開きます。
2. **Apps Gallery** または **Remote MCP Servers** で LinkUp を接続します。
3. API key を貼り付けます。
4. コネクタを有効化します。

## 無料プラン

- 一般的に **1,000 standard requests** と **100 deep research requests**。
