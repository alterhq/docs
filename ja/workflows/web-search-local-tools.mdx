---
title: "同梱の検索ツール"
description: "外部アカウント不要で Alter 同梱の Web 検索ツールを使う"
---

Alter には 2 つの Web 検索ツールが同梱されています。

これらは、ネイティブ Web 検索を持たないモデル（ローカルモデルを含む）を拡張する時に有効です。

<Info>
  Tavily、LinkUp、Firecrawl、Exa と違い、これらは Alter の利用に含まれており、外部 provider アカウントは不要です。
</Info>

## 比較

<CardGroup cols={2}>
  <Card title="Web Search (DuckDuckGo)" icon="compass">
    API key 不要。無料で即利用可能。snippets ベースなので高速ですが詳細は少なめです。
  </Card>
  <Card title="Web Search Gemini" icon="sparkles">
    Google search + Gemini 2.5 Flash。より詳細でソース付き。Alter Pro または Lifetime が必要です。
  </Card>
</CardGroup>

## 使い分け

- 無料アカウントや高速確認には **Web Search (DuckDuckGo)**。
- Pro/Lifetime で深い要約とソースが必要なら **Web Search Gemini**。

## 有効化

1. **Tools Manager** を開きます。
2. **Local Tools** に移動します。
3. **Web Search** と/または **Web Search Gemini** を有効化します。

<Tip>
  2 つを同時に常時有効化しないでください。タスクごとに 1 つを有効化するか、使う tool を明示してください。
</Tip>
