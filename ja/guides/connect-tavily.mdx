Tavily は Web リサーチ、抽出、crawling に適しています。

## Tavily の位置付け

- AI agents 向けの Web アクセス層
- search、extraction、research、crawling を 1 つの API で提供
- 最新 Web コンテキストによる grounding を重視

## 接続手順

1. **Tools Manager** を開きます。
2. **Remote MCP Servers** に移動します。
3. Tavily を追加します。
4. API key を貼り付けます。
5. コネクタを有効化します。
