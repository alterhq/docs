Exa は AI 向け Web 検索に特化した provider です。

## Exa の位置付け

- "The best search API for AI"
- 品質重視と benchmark 訴求
- search、contents、answer、crawl、research をカバー

## 接続手順

1. **Tools Manager** を開きます。
2. **Remote MCP Servers** または **Apps Gallery** で Exa を追加します。
3. API key を貼り付けます。
4. コネクタを有効化します。

## 無料プラン

- 一般的に **月 1,000 requests 前後** と案内されます。

<Warning>
  設定時に最新の料金と上限を確認してください。
</Warning>
