Firecrawl は大規模な構造化抽出に向いています。

## Firecrawl の位置付け

- "Turn websites into LLM-ready data"
- scraping、crawling、LLM 向けデータ整形を重視
- 動的サイトや本番運用を重視

## 接続手順

1. **Tools Manager** を開きます。
2. **Apps Gallery** に移動します。
3. **Firecrawl** を接続します。
4. API key を設定します。

## 無料プラン

- Firecrawl は無料プランを提供しています。

<Warning>
  最新のクォータ、レート制限、保持期間を確認してください。
</Warning>
