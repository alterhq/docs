Use esta página para fundamentar rapidamente o Alter com os arquivos e pastas importantes para sua tarefa.

## Adicione contexto do Finder ou de aplicativos

Você pode adicionar conteúdo arrastando arquivos ou pastas para o entalhe ou para o Hub. Você também pode usar **Clique com o botão direito > Adicionar para alterar**.

## Conteúdo suportado

<CardGroup cols={2}>
<Card title="Documents" icon="file-text">
`.xlsx`, `.docx`, `.pdf`, `.txt`, `.md`, `.org`, `.rtf`
</Card>
<Card title="Images" icon="image">
`.jpg`, `.png`, `.heic`, `.gif`, `.tiff`, `.svg`
</Card>
<Card title="Web content" icon="globe">
`.html`, URLs e texto arrastado
</Card>
<Card title="Email" icon="mail">
`.eml` e mensagens do Apple Mail
</Card>
<Card title="Audio" icon="music">
`.mp3`, `.wav`, `.m4a`, `.aac`, `.flac`
</Card>
<Card title="Code" icon="code">
Arquivos de origem, arquivos de configuração e scripts
</Card>
</CardGroup>

<Note>
Alter suporta descarte de diretório, OCR para imagens e PDFs e detecção automática de codificação de texto.
</Note>

## Transcrição de áudio e vídeo

Ao adicionar áudio ou vídeo, o Alter inicia a transcrição automaticamente.

<Steps>
<Step title="Drop one or more files">
Arraste os arquivos de mídia para o entalhe ou The Hub.
</Step>
<Step title="Wait for processing">
A transcrição começa automaticamente.
</Step>
<Step title="Ask targeted questions">
Experimente: “Resuma as principais decisões” ou “Liste os itens de ação por proprietário”.
</Step>
</Steps>

<Warning>
Para análise de imagens, use um modelo com capacidade de visão.
</Warning>
