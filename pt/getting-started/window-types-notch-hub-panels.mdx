### Conceito Geral

Alter fornece duas interfaces principais para interagir com IA:

<AccordionGroup>
  <Accordion icon="layout-dashboard" title="The Hub">
    A interface da janela principal com guias, barra lateral e gerenciamento abrangente do espaço de trabalho. Melhor para:
    - Conversas estendidas
    - Gerenciamento do espaço de trabalho
    - Multitarefa com vários contextos
    - Revisão de transcrições de reuniões
  </Accordion>
  
<Accordion icon="message-circle" title="Quick Panel">
    O painel flutuante leve acessível a partir do entalhe ou tecla de atalho. Perfeito para:
    - Consultas rápidas
    - Comandos de voz
    - Adições rápidas de contexto
    - Interações rápidas
  </Accordion>
</AccordionGroup>

## Interface do Hub

<Frame>
  O Hub está dividido em três seções principais:
  
1. **Barra lateral** – Navegação para conversas, áreas de trabalho, transcrições e muito mais
  2. **Área de conteúdo principal** - Interface com guias mostrando o conteúdo ativo
  3. **Área de entrada** - Caixa de prompt e gerenciamento de contexto
</Frame>

### Abrindo o Hub

<Steps>
  <Step title="Keyboard shortcut">
    Pressione `Shift + Cmd + H`
  </Step>
  <Step title="Menu bar">
    Selecione "Open The Hub" no ícone da barra de menu Alter
  </Step>
  <Step title="From a conversation">
    Abra um link de conversa em um painel flutuante
  </Step>
</Steps>

### Tipos de guias

<CardGroup cols={3}>
  <Card title="Conversations" icon="message-square">
    Sessões de chat com modelos de IA
  </Card>
  <Card title="Workspaces" icon="folder-open">
    Coleções de arquivos, pastas e aplicativos
  </Card>
  <Card title="Transcripts" icon="file-audio">
    Transcrições e gravações de reuniões
  </Card>
  <Card title="Calendar Events" icon="calendar">
    Reuniões e eventos agendados
  </Card>
  <Card title="Scheduled Jobs" icon="clock">
    Tarefas e fluxos de trabalho automatizados
  </Card>
</CardGroup>

### Atalhos de teclado

| Atalho | Ação |
|----------|--------|
| `Cmd + B` | Alternar visibilidade da barra lateral |
| `Cmd + W` | Fechar aba ativa |
| `Cmd + Option + →` | Próxima aba |
| `Cmd + Option + ←` | Aba anterior |
| `Cmd + K` | Limpar todos os contextos |
| `Cmd + Delete` | Excluir conversa ativa |

### Gerenciando Contextos

<Steps>
  <Step title="Add context">
    Arraste e solte arquivos ou use `@` na caixa de prompt
  </Step>
  <Step title="Navigate">
    Use as teclas de seta para navegar entre os contextos
  </Step>
  <Step title="Clear">
    Pressione `Cmd + K` para remover todos os contextos
  </Step>
</Steps>

### Painel do Inspetor

<CardGroup cols={2}>
  <Card title="Conversation Details" icon="info">

  </Card>
  <Card title="Context Management" icon="layers">

## Interface do painel rápido

O Painel Rápido (acessível a partir do entalhe) é dividido em três seções:

<CardGroup cols={3}>
  <Card title="1. Context" icon="layers">
    Seus arquivos, aplicativos e referências selecionados
  </Card>
  <Card title="2. Prompt" icon="edit">
    Onde você digita ou fala sua solicitação
  </Card>
  <Card title="3. Additional Selection" icon="plus">
    Acesso rápido a modelos e ações
  </Card>
</CardGroup>

### Navegação

Use atalhos de teclado no Painel Rápido:
- `←`, `↑`, `→`, `↓` - Navegue entre objetos
- `Enter` - Selecione um objeto
- `@` - Adicione elementos ao contexto
- `#` - Mostrar histórico de conversas
- `>` - Mostrar histórico de reuniões
- `/` - Liste e escolha o modelo
- `?` - Mostrar menu de ajuda

<Note>
  Consulte a seção **Caixa de prompt** abaixo para obter mais atalhos de teclado.
</Note>

### Navegação pelo teclado

| Chave | Ação |
|-----|--------|
| `←` `↑` `→` `↓` | Navegue pelos contextos |
| `Space` | Adicionar/remover selecionados |
| `Cmd + K` | Remover todos os contextos |

### Usando sua área de transferência

Cole o conteúdo diretamente: `Cmd + V`

O conteúdo da sua área de transferência é adicionado automaticamente ao contexto do Alter.

## A caixa de aviso

A **Caixa de Prompt** é onde a mágica acontece.

<Frame>
  Comece a digitar seu prompt ou um título Alter Action para enviá-lo ao seu modelo de IA.
</Frame>

### Atalhos de teclado

| Atalho | Ação |
|----------|--------|
| `Enter` | Enviar prompt com modelo ativo |
| `Cmd + Enter` | Pesquise na web (via Gemini) |
| `Shift + Enter` | Nova linha |
| `@` | Adicionar elementos ao contexto |
| `#` | Mostrar histórico de conversas |
| `>` | Mostrar histórico de transcrição de reuniões |
| `/` | Liste e escolha o modelo |
| `?` | Mostrar menu de ajuda |

<Tip>
  **Invocar Ações:** Digite qualquer caractere no título de uma Ação para ativá-la!
</Tip>

### Redimensionando janelas

  <Card title="Width" icon="move-horizontal">

  </Card>

<Info>
  **Útil para:** trabalho lado a lado, revisão de documentos longos, compartilhamento de tela, diferentes configurações de exibição

### Contínua fala para prompt

Tenha conversas de voz prolongadas sem tocar no teclado:

<Steps>
  <Step title="Hold voice hotkey">
    Pressione e segure `Fn` (ou sua tecla configurada)
  </Step>
  <Step title="Speak naturally">
    Diga sua solicitação em voz alta
  </Step>
  <Step title="Release">
    Solte para enviar para conversa
  </Step>
  <Step title="Keep going">
    Cada prompt de voz é adicionado ao mesmo tópico
  </Step>
</Steps>

<Tip>
  **Fluxo de trabalho sem usar as mãos:** Perfeito para brainstorming, ditado ou quando suas mãos estão ocupadas.
</Tip>

## Histórico de conversas

### Gerenciando histórico de bate-papo

Acesse o histórico digitando `#` na caixa de prompt ou usando a barra lateral do Hub.

<Frame>
  ![Histórico com tags](/images/other/history-tags.webp)
</Frame>

### Organizando com tags

Adicione tags como `#personal` para categorizar os bate-papos:

```
# #personal #food
```

Pesquise conversas que correspondam a várias tags.

### Renomeando conversas

<Steps>
  <Step title="Right-click">
    Clique com o botão direito em qualquer conversa da lista
  </Step>
  <Step title="Select Rename">
    Escolha `Rename` no menu
  </Step>
  <Step title="Type new name">
    Dê um nome descritivo
  </Step>
</Steps>

### Exportando conversas

Exporte como redução para documentação ou compartilhamento:

<Steps>
  <Step title="Find conversation">
    Localize-o na história
  </Step>
  <Step title="Right-click">
    Abra o menu de contexto
  </Step>
  <Step title="Select Export">
    Escolha a opção Exportar
  </Step>
  <Step title="Save">
    Escolha o local e salve como arquivo .md
  </Step>
</Steps>

<Card title="Export Uses" icon="download">
  - Documentação
  - Compartilhando com a equipe
  - Arquivando discussões importantes
  - Usando em outras ferramentas
</Card>
