Use espaços de trabalho quando descartes únicos de arquivos não forem suficientes e você precisar de contexto persistente.

## Por que usar espaços de trabalho

<CardGroup cols={3}>
<Card title="Faster responses" icon="zap">
Mantenha o contexto relevante indexado e pronto.
</Card>
<Card title="Lower cost" icon="credit-card">
Envie sempre contexto focado em vez de grandes entradas brutas.
</Card>
<Card title="Large folders" icon="folder-open">
Trabalhe com bases de código e conjuntos de documentos maiores.
</Card>
</CardGroup>

## Crie seu primeiro espaço de trabalho

<Steps>
<Step title="Open The Hub">
Pressione `Shift + Cmd + H`.
</Step>
<Step title="Go to Workspaces">
Abra a seção Espaços de trabalho na barra lateral.
</Step>
<Step title="Create workspace">
Clique em `+` no seletor de espaço de trabalho.
</Step>
<Step title="Add files and folders">
Adicione apenas os recursos necessários para um fluxo de trabalho executado com frequência.
</Step>
</Steps>

## Como os arquivos são preparados

| Tipo de arquivo | Conversão |
|---|---|
| PDFs | Markdown com marcadores de página |
| DOCX | Remarcação |
| XLSX | Markdown ou CSV |
| Imagens | Extração de texto OCR |
| Diretórios | Processamento recursivo |

<Note>
Arquivos de áudio e vídeo não são suportados diretamente nos espaços de trabalho. Transcreva-os primeiro, colocando-os no Alter.
</Note>

## Armazenamento local

Os dados do espaço de trabalho são armazenados localmente em:

```text
~/Library/Application Support/Alter/Workspaces/
```
