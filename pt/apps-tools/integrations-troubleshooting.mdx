Use esta página quando as integrações estiverem conectadas, mas não se comportarem conforme o esperado.

## Verificações rápidas

1. Abra novamente o **Gerenciador de ferramentas** e confirme se a ferramenta relevante está habilitada.
2. Confirme se o modelo selecionado suporta ferramentas.
3. Teste novamente com um simples prompt antes de executar fluxos de várias etapas.

## Problemas comuns

### A ferramenta não funciona

- Verifique se os campos de entrada obrigatórios estão presentes no seu prompt.
- Desative ferramentas não relacionadas para reduzir a ambigüidade de roteamento.
- Autentique novamente a integração se ela usar credenciais externas.

### O assistente responde sem usar ferramentas

- Mude para um modelo compatível com ferramentas.
- Pergunte com uma solicitação de ação explícita (por exemplo, “criar”, “enviar”, “adicionar”).

### A conexão parece válida, mas as chamadas falham

- Verifique o escopo da conta e as permissões do aplicativo no provedor.
- Tente novamente com uma ação de teste mínima para isolar a etapa com falha.
