---
title: "Espere"
description: "Pausar a execução entre ações de uso do computador"
---

Use Aguardar para adicionar intervalos de tempo entre as ações da interface do usuário quando páginas, caixas de diálogo ou estados de aplicativos precisarem de tempo para serem atualizados.

## O que faz

- Pausa a execução por um determinado número de segundos.
- Aceita valores numéricos e impõe um intervalo válido.

## Entrada

- `seconds` (obrigatório): número positivo de segundos para pausa (máx. 3600).

## Uso típico

- Adicione uma breve espera após a navegação.
- Faça uma pausa após enviar/clique antes de ler o contexto novamente.
- Estabilize fluxos de várias etapas onde a UI é renderizada de forma assíncrona.
