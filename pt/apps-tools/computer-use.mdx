O uso do computador fornece ao Alter uma camada de execução de UI para fluxos de trabalho de desktop. Ele pode inspecionar janelas, interagir com elementos da UI e acompanhar fluxos de várias etapas.

## Conjunto de ferramentas

- [Obter contexto do aplicativo ativo](/apps-tools/computer-use-get-active-app-context)
- [Executar](/apps-tools/computer-use-perform)
- [Espere](/apps-tools/computer-use-wait)
- [Rolar](/apps-tools/computer-use-scroll)

## Fluxo típico

1. Leia o estado atual da IU com **Obter contexto do aplicativo ativo**.
2. Acione ações de UI com **Executar**.
3. Use **Rolar** quando o elemento de destino estiver fora da tela.
4. Adicione **Aguarde** entre as etapas de carregamento ou transições de estado.
