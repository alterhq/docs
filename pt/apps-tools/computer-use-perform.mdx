---
title: "Executar"
description: "Execute ações de clique, preenchimento e atalho em elementos da UI"
---

Use esta ferramenta para executar interações de UI concretas depois de identificar alvos com Get Active App Context.

## Ações suportadas

- `click`: clique em um elemento de destino.
- `fill`: insira texto em um campo de texto ou área de texto.
- `shortcut`: envia atalhos de teclado (por exemplo `cmd+c`, `ctrl+shift+t`).

## Entradas

- `elementID` (obrigatório para `click` e `fill`): ID do elemento da saída Get Active App Context.
- `text` (obrigatório para `fill`): valor a ser inserido na entrada alvo.
- `shortcut` (obrigatório para `shortcut`): string de atalho.
- `windowID` (opcional): direcione uma janela específica (`pid/title`) em vez da janela ativa.

## Notas

- Os IDs dos elementos expiram com alterações na interface do usuário e no tempo da sessão. Releia o contexto se um elemento não for mais válido.
- Para direcionamento de janela em segundo plano, use o mesmo `windowID` em todas as etapas de leitura de contexto e execução.
