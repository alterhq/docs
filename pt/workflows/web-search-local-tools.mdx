O Alter inclui duas ferramentas de busca web prontas para uso.

Essas ferramentas ajudam quando voce usa modelos sem busca web nativa, incluindo modelos locais.

<Info>
  Diferente de Tavily, LinkUp, Firecrawl e Exa, estas ferramentas ja estao inclusas no uso do Alter e nao exigem conta em provedor externo.
</Info>

## Comparacao

<CardGroup cols={2}>
  <Card title="Web Search (DuckDuckGo)" icon="compass">
    Sem chave de API. Gratuito e imediato. Usa snippets, com resposta rapida e menos detalhada.
  </Card>
  <Card title="Web Search Gemini" icon="sparkles">
    Usa Google search com Gemini 2.5 Flash. Respostas mais detalhadas e com fontes. Exige assinatura Alter Pro ou Lifetime.
  </Card>
</CardGroup>

## Quando usar cada um

- Use **Web Search (DuckDuckGo)** no plano gratuito ou para buscas rapidas.
- Use **Web Search Gemini** com Pro/Lifetime para mais profundidade e fontes.

## Como ativar

1. Abra **Tools Manager**.
2. Va para **Local Tools**.
3. Ative **Web Search** e/ou **Web Search Gemini**.

<Tip>
  Nao deixe dois tools de busca ativos por padrao. Ative um por tarefa ou adicione instrucao explicita de qual tool usar.
</Tip>
