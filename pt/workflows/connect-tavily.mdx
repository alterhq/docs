Tavily e uma opcao forte para pesquisa web, extracao e crawling.

## Como Tavily se posiciona

- Camada de acesso web para agentes de IA
- API unica para search, extraction, research e crawling
- Foco em grounding com contexto web atualizado

## Como conectar

1. Abra **Tools Manager**.
2. Entre em **Remote MCP Servers**.
3. Adicione Tavily.
4. Cole a chave de API.
5. Ative o conector.
