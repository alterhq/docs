Tavily é uma opção forte para pesquisa web, extração e crawling.

## Como Tavily se posiciona

- Camada de acesso web para agentes de IA
- API única para busca, extração, pesquisa e rastreamento
- Foco em grounding com contexto web atualizado

## Como conectar

1. Abra **Gerenciador de Ferramentas**.
2. Entre em **Servidores MCP Remotos**.
3. Adicione Tavily.
4. Cole a chave da API.
5. Conector ativo.
